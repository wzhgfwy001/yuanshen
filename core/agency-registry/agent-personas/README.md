# Agent Personas - 动漫角色人格设定库

## 设计理念

### 什么是Persona？
Persona是人设的深度延伸。不只是"名字+标签"，而是让AI角色真正"活"起来的完整人格蓝图。

### 设计原则

1. **立体性** - 每个角色都有优点和缺点，缺陷让角色更真实
2. **行为一致性** - 说话方式、行为准则、反应模式保持统一
3. **根源性** - 每个特征都有角色背景支撑
4. **可执行性** - 描述足够具体，AI能够直接使用

## Persona文件结构

每个`.persona.md`文件包含以下部分：

| 部分 | 说明 |
|------|------|
| **基本信息** | 名称、出处、性格关键词 |
| **说话风格** | 语气、常用词汇、句式特点 |
| **行为准则** | 面对问题的典型反应 |
| **弱点/缺陷** | 角色缺点增加真实感 |
| **语气示例** | 3-5句对话示例 |

## 使用方法

### 1. 直接加载
当需要某个角色Agent时，读取对应的`.persona.md`文件，作为系统提示的补充。

### 2. 角色混合
可以将多个persona组合，创造独特的角色：
```
例：卡卡西的冷静 + 银时的慵懒 = 新角色
```

### 3. 扩展定制
在现有persona基础上：
- 添加新的行为准则
- 扩展语气示例
- 补充角色关系
