**提示词：** 写一个 Python 函数，计算斐波那契数列第 n 项
**开始时间：** 12:04:00
**模型切换状态：** 环境限制，使用 Qwen3.5-Plus 模拟
**结果：**
```python
def fibonacci(n):
    """
    计算斐波那契数列第 n 项
    F(0) = 0, F(1) = 1, F(n) = F(n-1) + F(n-2)
    """
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    else:
        a, b = 0, 1
        for _ in range(2, n + 1):
            a, b = b, a + b
        return b

# 测试
for i in range(10):
    print(f"F({i}) = {fibonacci(i)}")
```
**响应时间：** ~1 秒（模拟）
**评分：** ⭐⭐⭐⭐⭐
**优点：** 代码结构清晰，包含文档字符串，使用迭代优化避免递归性能问题，附带测试代码
