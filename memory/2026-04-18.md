## 22:25 两个新任务完成 + GitHub网络问题

### ✅ 任务1：扩展映射逻辑到所有category

**修改文件：** core/subagent-manager/mapping-loader.js

**扩展内容：**
- 原逻辑：只处理 `specialized` 分类
- 新逻辑：支持所有 category + 通配符 `*`

**新 getMappedCategory 逻辑：**
```javascript
// 1. 先检查原始 category 是否有映射
if (_categoryRemap[originalCategory]) {
  const remap = _categoryRemap[originalCategory];
  if (agentName && remap[agentName]) {
    return { mappedCategory: remap[agentName], wasMapped: true, originalCategory, agentName };
  }
}

// 2. 再检查通配符 * category
if (_categoryRemap['*'] && agentName && _categoryRemap['*'][agentName]) {
  return { mappedCategory: _categoryRemap['*'][agentName], wasMapped: true, mappedFrom: '*' };
}

// 3. 无映射，返回原分类
return { mappedCategory: originalCategory, wasMapped: false };
```
