## 🌍 第一部分：世界观核心

### 1.1 时代背景
- **时间：** 3026年，太阳系开发成熟
- **势力：** 东方联合国军 / 西方联合国军 / 各国星际舰队 / 星际海盗
- **人类分布：** 有人类在地球 / 有人类出生在外太空
- **文化差异：** 各势力理念不同，需要维稳

### 1.2 宇宙起源
- 大爆炸后外围生命向中心逃离，地球是能量补充站
- 九层宝塔位于昆仑山脉
- 灵魂能量是宇宙顶级能量

### 1.3 灵魂能量（钥匙理论）
> 灵魂能量 = 撬动宇宙奥秘的钥匙

- **每个人的灵魂体质不同 = 钥匙不同 = 能力不同**
- **能力上限 = 与对应宇宙规则的契合程度**
- 法术/魔法 = 灵魂能量的不同运用方式

### 1.4 宝塔与地球生命进化
- 宝塔抵达时生命薄弱 → 需要培养开智
- 植物 → 动物 → 人类 = 灵魂能量越来越精纯
- 人类站在地球之巅

- 大爆炸后外围幸存的生命体
- 以纯天魂形态存于天宫
- 可申请投胎转世

| 魂 | 来源 | 作用 |
|----|------|------|
| 天魂 | 遗民灵魂投胎 | 带来潜能 |
| 地魂 | 审判洗礼 | 因果清算 |
| 人魂 | 家族气运 | 家族传承 |

### 三大闭环
1. **投胎闭环：** 天魂+地魂+人魂 → 新生命
2. **审判闭环：** 大恶供能 / 大善聪慧 / 普通轮回

- 触发：被人拿宁千阳威胁宁家，置于危险境地
- 效果：姐弟心灵感应共振，能力共鸣放大，双双S+级

### 凌午夜觉醒路径
```
第一阶段（6-7岁）：肉体强化
    ↓ 危险境地
```
