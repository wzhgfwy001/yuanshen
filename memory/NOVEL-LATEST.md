## 🌍 世界观核心

### 时代背景
- **时间：** 3026年，太阳系开发成熟
- **势力：** 东方联合国军 / 西方联合国军 / 星际舰队 / 星际海盗

### 灵魂能量（钥匙理论）
- 灵魂能量 = 撬动宇宙奥秘的钥匙
- 每个人钥匙不同 = 能力不同
- 能力上限 = 与宇宙规则的契合程度

### 九层宝塔
太虚宫 / 苍穹殿 / 天策府 / 星辰海 / 人间阁 / 轮回殿 / 因果簿 / 缉魂司 / 阎罗殿

### 三魂体系
- 天魂（遗民投胎带来潜能）
- 地魂（审判洗礼因果清算）
- 人魂（家族气运传承）

## 👥 学院小队（7人）

### 凌午夜（主角）
```
【性格发展】
├── 前期（1-100章）：路飞式乐天派
│   ├── 天真无畏，"我要成为最强的！"
│   ├── 对危险没概念往前冲
│   ├── 绝对信任伙伴
│   ├── "我们是伙伴啊！"
│   └── 打架是交朋友的方式
│
├── 中期（101-200章）：转变中
│   ├── 同伴第一次重伤，开始意识到责任
│   ├── 笑容变少
│   ├── 战斗时不再莽撞
│   └── 但乐观没变
│
└── 后期（201-300章）：沉稳但不变
    ├── 沉稳内敛但不失幽默
    ├── 决策果断但会听取意见
    ├── 乐观勇敢探索精神一直都在
    └── 真正的领袖气质

【能力】时空操控（逐步觉醒）
【定位】队伍核心/领袖
```
