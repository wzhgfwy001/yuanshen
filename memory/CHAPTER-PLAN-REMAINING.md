# 《吾名午夜》地球篇 - 章节规划

**版本：** v2.0  
**理念：** 有血有肉，真情实感，热血浪漫且勇敢  
**原则：** 不限章节，让故事自然发展

---

## 📖 核心理念

### 每个角色都需要

| 角色 | 需要体现的 |
|------|-----------|
| **凌午夜** | 路飞式乐天→沉稳领袖的转变，亲情友情爱情，守护同伴的决心 |
| **北冥瑶** | 腹黑日常→被折服→暗恋→自卑→接受→恋爱，完整感情线 |
| **西门吹雪** | 傲娇日常，被调侃美貌的无奈，男扮女装的经历，与父亲的和解 |
| **东方凌薇** | 炫富背后的脆弱，父母的牵挂，对凌午夜的感情 |
| **中原星辰** | 中二的浪漫，对南宫墨的暗恋，平民的奋斗 |
| **南宫墨** | 冷淡外表下的温柔，家族的秘密，被守护的感动 |
| **三哥** | 神秘背后的故事，健身的执着，"我很帅"的自信，万事通的能力 |
| **宁千阳** | 舔狗的日常，对凌飞雪的真心，关键时刻的成长 |

## 📊 角色成长与结局

### 凌午夜
```
起点：6岁，路飞式乐天派
终点：21岁，沉稳领袖

成长：
├── 同伴受伤→开始转变
├── 保护同伴→责任感

├── 守护伙伴→真正的领袖
└── 但乐观勇敢探索精神一直都在

感情：北冥瑶 ❤️（地球结尾确认）
```
