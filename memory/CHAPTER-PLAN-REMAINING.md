## 📖 写作风格指导

### 场景描写
```
不要：天空很蓝
要写：清晨的阳光穿透云层，在昆仑山的峰顶洒下金色的光芒。远处的云海翻涌，像是一片无边无际的白色海洋，让人仿佛置身于仙境之中。

不要：她很漂亮
要写：北冥瑶笑起来的时候，眼睛会弯成两道月牙，嘴角微微上扬，带着一丝说不清道不明的韵味。让人看了就忍不住想知道她在笑什么。

不要：西门吹雪很帅
要写：一头银发如瀑布般垂落，剑眉星目，轮廓分明得像是刀刻出来的一样。站在阳光下，那张脸精致得连女人都会嫉妒。但他自己似乎完全不这么认为，反而一脸不耐烦。
```

### 内心描写
```
不要：他很紧张
要写：凌午夜站在天道学院的门口，手心微微出汗。他攥紧了背包的带子，深吸一口气。六年了，从那个襁褓中的婴儿，到如今站在这里的新生。六年里，他无数次幻想过这个场景。但真正站在这里的时候，他的心跳得比想象中快得多。

不要：北冥瑶喜欢他
要写：北冥瑶坐在窗边，手里捧着一本书，目光却不时飘向正在练习空间移动的凌午夜。她看着他在失败中一次次站起来，看着他摔倒了就笑着爬起来。这种不服输的劲儿……真是个傻子。但不知道为什么，看着看着，嘴角就忍不住上扬了。
```

### 人物刻画重点

#### 凌午夜（路飞式乐天）
```
日常：
├── 笑容满面，像个小太阳
├── "我们是伙伴！"
├── 打架是为了交朋友
├── 饿了就吃，困了就睡
├── 对危险完全没概念
└── 但关键时刻非常可靠

内心：
├── 其实很想念妈妈
├── 想让家人骄傲
├── 把每个伙伴都当家人
└── 暗暗努力但不说
```

#### 北冥瑶（腹黑温柔）
```
日常：
├── 表面温柔体贴
├── 喜欢整人但无恶意
├── 笑起来甜甜的
├── 但不知道她在想什么
└── 偶尔露出腹黑一面

内心：
├── 身世困扰她
├── 渴望被理解
├── 第一次对人心动
└── 不知道怎么处理感情
```

#### 西门吹雪（傲娇剑痴）
```
日常：
├── 面瘫话少
├── "你们不懂剑"
├── 被夸帅就烦
├── "我是认真的在讨论剑"
└── 但其实很在意同伴

内心：
├── 对剑的热爱超过一切
├── 背负父亲的罪
├── 不善言辞但很在乎伙伴
└── 被调侃时其实很无奈
```

#### 东方凌薇（话痨炫富）
```
日常：
├── 说话停不下来
├── "我爷爷有钱！"
├── 方向感差到离谱
├── 爱吃甜食
└── 给发明起奇怪名字

内心：
├── 用吵闹掩盖不安
├── 父母不在很孤独
├── 对凌午夜有好感
└── 怕被当成大小姐
```

#### 中原星辰（中二热血）
```
日常：
├── 说话大声
├── "我的目标是！"
├── 容易激动
├── 崇拜强者
└── 和凌午夜并称活宝

内心：
├── 平民出身自卑
├── 努力型选手
├── 喜欢南宫墨但不敢说
└── 把凌午夜当最好的兄弟
```

#### 南宫墨（冷淡理性）
```
日常：
├── 话少但句句精准
├── 总是看书
├── "我对这没兴趣"
├── 偶尔露出微笑
└── 腹黑的一面

内心：
├── 能读懂别人想法
├── 其实很孤独
├── 习惯一个人
├── 被中原星辰感动
└── 慢慢打开心扉
```

#### 三哥（神秘健身狂）
```
日常：
├── 总是健身
├── "健完身我很帅的"
├── "我可是用枪的"
├── 话不多但很可靠
└── 万事通

内心：
├── 记得前世的事
├── 一直在锻炼肉体
├── 想变得更强
└── 把伙伴当家人
```
