## 新分数规则

### 冲稳保分数范围 & 概率

| 类别 | 分数范围 | 概率范围 | 说明 |
|------|----------|----------|------|
| **冲** | 考生成绩 +5~+15分（含） | 50%~30% | 分数越高概率越低 |
| **稳** | 考生成绩 ±5分（含） | 50%~80% | 分数越高概率越高 |
| **保** | 考生成绩 -5~-20分（含） | 80%~100% | 分数越低概率越高 |

### 概率计算公式

```javascript
// 冲区间：+5分→50%，+15分→30%
if (diff >= 5 && diff <= 15) {
  probability = 50 - (diff - 5) * 2;
}

// 稳区间：-5分→50%，+5分→80%
if (diff >= -5 && diff < 5) {
  probability = 50 + (diff + 5) * 3;
}

// 保区间：-5分→80%，-20分→100%
if (diff >= -20 && diff < -5) {
  probability = 80 + (diff + 5) * (4/3);
}
```

```javascript

```

## 前端改动要求

- 前端需传入 `userScore` 参数（考生成绩）
- 不再需要传入 `minScore`/`maxScore`
- 云函数内部自动计算查询范围
