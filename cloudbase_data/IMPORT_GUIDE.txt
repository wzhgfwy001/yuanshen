## 导入步骤
1. 打开微信开发者工具
2. 点击"数据库" -> "benke"集合
3. 点击"导入"按钮
4. 选择 benke.json 文件
5. 导入格式选择: JSON
6. 点击确认

## 验证导入成功
导入后，在数据库控制台执行以下查询验证：
db.benke.countDocuments()  // 应该返回 21425
db.zhuanke.countDocuments()  // 应该返回 11912
