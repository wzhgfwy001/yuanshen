# 天元学院校服与院徽设定

**版本：** v1.0
**创建：** 2026-04-30
**状态：** 初始设定

---

## 📋 总览

天元学院分为三院：
- **天道院** - 特殊天赋（时空规则，暂不公开）
- **太玄院** - 精神/元素/秘法
- **天枢院** - 武道/战技/科技

每院有专属：
1. 院徽（徽章）
2. 校服特色（颜色、款式细节、配饰）

---

## 🎨 基础校服规格

### 全院统一
| 项目 | 规格 |
|------|------|
| **主色调** | 蓝色（天元学院基础色） |
| **校服款式** | 立领长袍+束腰腰带 |
| **材质** | 灵力纤维（可防护） |
| **左胸** | 天元学院校徽 |
| **右胸/后背** | 各分院院徽 |

## 🏫 各分院详情

### 天道院（特殊天赋院）

#### 院徽设计
```
     ★
   ╱ ✧ ╲
  ╱  ◇  ╲
 ╱   ⚝  ╲
═════════
```

| 元素 | 含义 |
|------|------|
| 中心六芒星 | 宇宙星辰，内外宇宙交汇 |
| 六芒星外的圆环 | 时间与空间的循环 |
| 圆环上下四颗小星 | 四象/四方守护 |
| 银色底纹 | 神秘、高贵 |

**徽章颜色：** 银白底 + 淡蓝光纹

---

#### 校服特色

| 项目 | 描述 |
|------|------|
| **主色** | 银白 + 淡蓝光纹 |
| **领口** | 立领，领口有银色流苏 |
| **袖口** | 宽袖，袖口有星光刺绣 |
| **腰带** | 银丝腰带，腰带扣是六芒星 |
| **下摆** | 裙摆有星空暗纹 |
| **特点** | 最华丽、最神秘，像夜空 |

**标志性装饰：**
- 银色发冠（统一配发）
- 星光纹样的披风（外出历练时穿）

| 项目 | 描述 |
|------|------|
