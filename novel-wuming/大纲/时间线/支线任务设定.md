## 🗃️ 任务详解

### 任务类型一：降妖采集灵力

#### 任务背景
> 飞升塔需要持续的灵魂能量维持运转和遗民永生。地球上的生命死亡后，灵魂能量需要被定期收集。小队协助飞升塔完成能量采集任务。

#### 任务流程
```
野外死亡生命 → 灵魂残留（野鬼/游魂）
    ↓
飞升塔探测到残留能量
    ↓
小队前往降服+回收
    ↓
能量注入飞升塔存储系统
```

#### 任务清单

| 任务编号 | 任务名称 | 目标 | 揭示内容 | 章节 |
|----------|----------|------|----------|------|
| B-001 | 深山古墓怨灵 | 降服百年怨灵 | 灵魂能量守恒定律 | 31-35 |
| B-002 | 矿洞灵脉守护 | 采集矿脉灵力 | 自然界的灵魂能量循环 | 36-40 |
| B-003 | 战乱遗址亡魂 | 引导亡魂安息 | 战争遗留灵魂处理 | 41-45 |
| B-004 | 妖兽内丹回收 | 获取妖兽灵核 | 动物灵魂能量的特殊性 | 46-50 |

#### 揭示内容

**灵魂能量守恒定律：**
```
生命死亡 → 灵魂释放 → 如果不被回收 → 凝聚成怨灵/野鬼
    ↓
怨灵/野鬼会吸收其他弱小灵魂 → 越来越强
    ↓
最终形成恶灵 → 必须消灭
    ↓
被消灭时能量消散 → 回归自然
```

**能量回收机制：**
- 飞升塔有探测装置，可以感知一定范围内的灵魂能量波动
- 小队携带回收器，类似探测器
- 回收的能量通过特定方式注入飞升塔

### 任务类型二：协助地府除魔

#### 任务背景
> 飞升塔的地府系统（缉魂司、轮回殿、因果簿等）需要处理大量灵魂。地府工作人员偶尔会人手不够，需要外部协助。小队在此类任务中逐渐了解地府的运作机制。

#### 地府组织架构（揭示）

```
┌─────────────────────────────────────────────┐
│                    飞升塔                    │
│                  （九层结构）                 │
├─────────────────────────────────────────────┤
│ 第九层：太虚宫（三清殿 + 四帝宫）→ 最高决策层 │
│ 第八层：苍穹殿（诸天大神）→ 诸神管理         │
│ 第七层：天策府 → 攻伐防御                     │
│ 第六层：星辰海（日月星辰诸神）→ 天象管理     │
│ 第五层：人间阁 → 城隍、山神、河神           │
│ 第四层：轮回殿 → 投胎名单管理               │
│ 第三层：因果簿 → 善恶簿记录                 │
│ 第二层：缉魂司 → 拘魂引导                   │
│ 第一层：阎罗殿 → 最终审判                   │
└─────────────────────────────────────────────┘
         ↓
【地府核心系统】
├── 缉魂司：负责在人死后拘押灵魂，引导至审判
├── 轮回殿：管理投胎名单，决定去向
├── 因果簿：记录善恶，作为审判依据
└── 阎罗殿：最终审判，决定刑罚或投胎
```

#### 任务清单

| 任务编号 | 任务名称 | 目标 | 揭示内容 | 章节 |
|----------|----------|------|----------|------|
| C-001 | 迷失灵魂引路 | 引导游魂回归 | 缉魂司的引路机制 | 31-35 |
| C-002 | 恶鬼越狱事件 | 镇压越狱恶鬼 | 地府关押系统 | 36-40 |
| C-003 | 投胎名单核对 | 协助核对名单 | 轮回殿投胎规则 | 41-45 |
| C-004 | 因果簿异常 | 调查记录错误 | 因果簿运作方式 | 46-50 |

#### 揭示内容

**缉魂司引路机制：**
```
人死亡 → 灵魂自动飘向最近的地府入口
    ↓
缉魂司工作人员（鬼差）交接
    ↓
登记信息 → 引导至审判殿
    ↓
【问题】如果灵魂迷路/被阻挡？
    ↓
需要外部协助引导 → 小队任务
```

**野鬼形成原因：**
```
正常：人死亡 → 缉魂司拘押 → 审判 → 投胎
    ↓
异常1：灵魂被执念困住 → 无法离开 → 原地徘徊 → 形成游魂
异常2：灵魂被恶意利用 → 被束缚 → 成为工具 → 形成鬼傀
异常3：灵魂被其他灵魂吞噬 → 失去意识 → 形成怨灵
```

**灵魂回收机制：**
```
步骤1：灵魂探测（缉魂司感知范围）
    ↓
步骤2：自动标记（异常灵魂会发出特殊波动）
    ↓
步骤3：人工介入（有问题的灵魂需要协助处理）
    ↓
步骤4：引导回归（正常灵魂）或强制拘押（异常灵魂）
```

### 任务类型三：除魔卫道

#### 任务背景
> 暗流组织的活动会导致一些异常灵魂/生物的产生。小队需要处理这些暗流留下的"烂摊子"，同时也在这些任务中逐渐了解暗流和吞噬者的关系。

#### 任务清单

| 任务编号 | 任务名称 | 目标 | 揭示内容 | 章节 |
|----------|----------|------|----------|------|
| D-001 | 寄生体清理 | 消灭暗流制造的寄生体 | 吞噬者的寄生机制 | 61-70 |
| D-002 | 邪气污染净化 | 净化被暗流污染的区域 | 暗流邪气的影响 | 71-80 |
| D-003 | 夺舍者追踪 | 追踪被夺舍的灵魂 | 夺舍原理 | 81-85 |
| D-004 | 暗流实验残骸 | 调查暗流实验 | 吞噬者的真正目的 | 86-90 |

#### 揭示内容

**吞噬者的寄生机制：**
```
吞噬者（纯灵魂形态）
    ↓
需要宿主才能在物质界活动
    ↓
找到合适的灵魂/肉体 → 夺舍/寄生
    ↓
寄生后保留宿主部分能力
    ↓
完全控制宿主身体
```

**夺舍原理：**
```
吞噬者侵蚀宿主灵魂
    ↓
逐渐压制原主人格
    ↓
最终完全替代
    ↓
【区别】
- 完全夺舍：原主人格彻底消失
- 部分夺舍：原主人格被压制但未消失
```

### 任务类型四：迷失指引（北冥瑶专属）

#### 任务背景
> 北冥瑶作为阴阳师后裔，天生具有与灵魂沟通的能力。她可以感知到迷失的灵魂，并帮助他们找到正确的方向。这类任务主要展现北冥瑶的能力和她温柔的一面。

#### 任务清单

| 任务编号 | 任务名称 | 目标 | 揭示内容 | 章节 |
|----------|----------|------|----------|------|
| E-001 | 迷路小孩灵魂 | 帮助走失孩童灵魂回家 | 阴阳师的感知能力 | 35-40 |
| E-002 | 战争亡魂超度 | 超度无法安息的战士灵魂 | 地府对接机制 | 41-45 |
| E-003 | 执念深重灵魂 | 化解特殊执念的灵魂 | 执念与灵魂的关系 | 46-50 |
| E-004 | 家族祖先联络 | 帮助活人联络已故祖先 | 因果簿查询功能 | 51-60 |

#### 揭示内容

**阴阳师感知能力：**
```
北冥瑶天生可以感知灵魂
    ↓
经过训练可以与灵魂沟通
    ↓
能力进阶后可以引导灵魂
    ↓
【专属任务】帮助迷失/迷路/执念的灵魂
```

**执念与灵魂的关系：**
```
灵魂残留的原因：
├── 执念：放不下某件事/某个人
├── 愧疚：做了某件无法原谅自己的事
├── 爱：为了某个人而坚持
├── 恨：被某人害死，执念报仇
└── 使命：还有事没做完

【处理方式】
├── 执念轻：引导即可
├── 执念重：需要化解执念才能引导
└── 执念极深：可能永久残留，成为厉鬼
```

## 📝 注意事项

1. **任务不宜过长**：每个支线任务1-3章内完成
2. **揭示要自然**：不要直接讲解，通过任务过程自然展示
3. **角色分工明确**：发挥每个角色的特长
4. **保持神秘感**：不要一次性揭示太多，留下悬念
5. **与主线呼应**：支线中的发现要能推动主线发展
