## 🏫 各学院详情

### 天元学院（华夏核心）
**定位：** 飞升塔遗民建立，表面是异能学院，实际是培养对抗暗流/吞噬者的核心力量

| 项目 | 内容 |
|------|------|
| 内部结构 | 天道院（特殊天赋）+ 太玄院（精神/元素）+ 天枢院（武道/科技） |
| 特殊地位 | 2组内定名额（可直接参加世界大赛） |
| 招生 | 表面公开招生，实际有筛选机制 |
| 校长 | 待定 |

**天道院特别说明：**
- 暂不公开时空规则
- 只收有特殊天赋的学生
- 学生对自己的能力来源并不完全清楚

---

### 樱花学院（樱花省）
**定位：** 阴阳师和忍者的传承地

| 项目 | 内容 |
|------|------|
| 位置 | 樱花省（已被华夏吞并的阴阳师领地） |
| 特色 | 阴阳师、忍术、式神操控 |
| 院长 | 待定（可能是阴阳师家族长老） |
| 历史 | 安倍晴明创立的阴阳师流派分支 |

**主要学科：**
- 阴阳道（式神、阴阳术）
- 忍术（暗杀、潜伏、情报）
- 傀儡术（机械/式神傀儡）

### 禅宗学院（中土省）
**定位：** 中原禅法正宗，佛门秘术传承

| 项目 | 内容 |
|------|------|
| 位置 | 中土省 |

## 🏆 比赛体系

### 华夏学院大比（每2年一届）

| 项目 | 内容 |
|------|------|
| 周期 | 每2年一次 |
| 参赛资格 | 各学院选送队伍 |
| 名额分配 | 除天元学院外，每院可选送5组队伍 |
| 选拔名额 | 10组队伍晋级世界学院大比 |
| 天元学院 | 2组内定 + 1组（二线选拔） |

**赛制：**
```
报名 → 海选（所有队伍）→ 淘汰赛 → 决赛
                                    ↓
                         决出10组晋级世界大赛
```

---

### 世界学院大比（每4年一届）

| 项目 | 内容 |
|------|------|
| 周期 | 每4年一次 |
| 参赛资格 | 各国选拔的冠军队伍 |
| 年龄限制 | 20岁以下 |
| 天元学院 | 3组参赛（2组内定+1组选拔） |

**赛制：**
```
入围赛（64支队伍）→ 小组赛 → 淘汰赛 → 决赛
                                          ↓
                                    冠军诞生
```

---

### 个人赛

**触发条件：** 团队赛10强决出后

**参赛资格：** 10强队伍的队长

**赛制：**
```
10强队长 → 循环积分赛 → 淘汰赛 → 决赛
                            ↓
                       个人赛冠军
```

## 🎖️ 补充建议

### 1. 比赛地点
| 比赛 | 地点选择 | 理由 |
|------|----------|------|
| 华夏大比 | 轮流制（每届不同学院承办） | 增加各学院参与感 |
| 世界大赛 | 中立地点（公海岛屿/中立国） | 公平性 |

### 2. 奖励设定
| 奖励类型 | 内容 |
|----------|------|
| 团队冠军 | 奖金 + 荣誉称号 + 修炼资源 |
| 个人冠军 | 额外奖励（如法器/秘籍） |
| 晋级奖励 | 每晋级一轮有相应奖励 |

### 3. 天元学院的特殊机制
| 机制 | 说明 |
|------|------|
| 天道院秘密参赛 | 天道院学生可用其他身份参赛，不暴露真实实力 |
| 番外小队 | 像三哥这种特殊身份角色，可作为“外援”临时加入 |

### 4. 其他国家/地区

| 非洲 | 部落学院 | 自然崇拜、部落秘术 |
