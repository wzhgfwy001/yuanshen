## 🏫 各学院详情

### 天元学院（华夏核心）
**定位：** 飞升塔遗民建立，表面是异能学院，实际是培养对抗暗流/吞噬者的核心力量

| 项目 | 内容 |
|------|------|
| 内部结构 | 天道院（特殊天赋）+ 太玄院（精神/元素）+ 天枢院（武道/科技） |
| 特殊地位 | 2组内定名额（可直接参加世界大赛） |
| 参赛名额 | 2组内定 + 3组参加选拔（与其他学院争夺8组名额） |
| 招生 | 表面公开招生，实际有筛选机制 |

---

## 🏆 比赛体系

### 华夏学院大比（每2年一届）

| 项目 | 内容 |
|------|------|
| 周期 | 每2年一次 |
| 参赛资格 | 各学院选送队伍 |
| 名额分配 | 除天元学院外，每院可选送5组队伍 |
| 天元学院 | 2组内定 + 3组参加选拔 |

**名额决出规则：**
```
天元学院：2组内定（直接晋级）
        ↓
剩下8组名额：
- 天元学院3组选拔队
- 其他学院5组 × 各学院
        ↓
共同争夺8组晋级名额
```

---

### 世界学院大比（每4年一届）

| 项目 | 内容 |
|------|------|
| 周期 | 每4年一次 |
| 参赛资格 | 各国选拔的冠军队伍 |
| 年龄限制 | 20岁以下 |
| 每组人数 | 6人 |
| 天元学院 | 3组参赛（2组内定+1组选拔） |

---

### 个人赛

**触发条件：** 团队赛10强决出后

**参赛资格：** 10强队伍的队长

**赛制：**
```
10强队长 → 循环积分赛 → 淘汰赛 → 决赛
```
