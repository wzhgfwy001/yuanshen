# 《吾名午夜》去AI味写作规范

**版本：** v1.0
**更新：** 2026-04-18
**依据：** 工作流7点要求第4条

---

## ⚠️ 为什么去AI味是强制要求

AI生成的文字有明显的"完美感"：
- 描写太标准、太套路
- 对话像在演讲，不够口语化
- 人物反应太一致，缺乏个性
- 情绪表达夸张，不够真实

**读者会出戏，觉得"假"。**

---

## 🔴 AI味特征清单（必须清除）

### 1. 描写类（禁用词）

| ❌ AI味表达 | ✅ 替代方案 |
|-------------|-------------|
| "眼中闪过...的光芒" | 具体眼睛反应：眯眼、眨眼、视线移动 |
| "心中涌起...的情绪" | 身体反应：心跳、呼吸、肌肉紧张 |
| "嘴角微微上扬/抽搐" | 具体动作：撇嘴、抿嘴、咬牙 |
| "脸色苍白/铁青" | 细节：冒冷汗、脸发热、手抖 |
| "周身散发着...的气息" | 周围环境反应：温度、气味、声音 |
| "完美的笑容" | 真实的不完美：笑得太开、笑得僵硬 |
| "英俊的脸庞" | 具体描写：下巴线条、眉毛形状 |

### 2. 动作类（禁用词）

| ❌ AI味表达 | ✅ 替代方案 |
|-------------|-------------|
| "他优雅地..." | 具体动作分解 |
| "动作行云流水" | 描述实际动作细节 |
| "仿佛...一般" | 直接描述 |
| "从容不迫" | 具体表现：手稳、呼吸均匀 |
| "毫不犹豫" | 犹豫的具体表现 |

### 3. 心理类（尽量少用）

| ❌ AI味表达 | ✅ 替代方案 |
|-------------|-------------|
| "他心想..." | 用动作/表情暗示 |
| "内心深处..." | 不用 |
| "涌起一股...的感情" | 身体反应 |

**原则：心理描写能删就删，用动作和对话暗示**

### 4. 对话类（问题）

| ❌ AI味表达 | ✅ 替代方案 |
|-------------|-------------|
| 完整的客套话 | 省略、寒暄压缩 |
| 过于有逻辑的表达 | 跳跃、打断、沉默 |
| 废话连篇 | 精简到只剩关键词 |
| 总是回应对方 | 不回应、转移话题、沉默 |

### 5. 反应类（问题）

| ❌ AI味表达 | ✅ 替代方案 |
|-------------|-------------|
| 总是恰当的反应 | 过度反应、冷漠、错愕 |
| 永远冷静/永远激动 | 偶尔失控、偶尔冷漠 |
| 从不犯错 | 犹豫、判断失误、后悔 |

## ✅ 正确写法示范

### 案例1：心理改动作

```markdown
❌ AI味：
凌午夜感到一阵恐惧，心跳加速，手心出汗。他想逃跑，但双腿却像灌了铅一样沉重。

✅ 去AI味：
凌午夜的呼吸停了一拍。
手心湿了。
他想跑。
腿不听使唤。
```

### 案例2：对话改沉默

```markdown
❌ AI味：
"我真的不是故意的，你相信我吗？"东方凌薇急切地解释，声音有些颤抖。
"我理解你的感受，但是..."凌午夜叹了口气。

✅ 去AI味：
"我..."
东方凌薇话没说完。
凌午夜已经转身走了。
```

### 案例3：完美改缺陷

```markdown
❌ AI味：
凌午夜冷静地分析了情况，迅速做出了正确的判断，带领队伍成功突围。

✅ 去AI味：
凌午夜脑子一片空白。
他不知道该怎么办。
腿有点软。
"走哪边？"他问。
没人回答。
```

### 案例4：环境描写简化

```markdown
❌ AI味：
月光洒在寂静的森林里，树叶在微风中沙沙作响，远处的山峦若隐若现，仿佛一幅水墨画。

✅ 去AI味：
树叶响了一声。
有什么东西在移动。
月光不够亮。
```

### 案例5：战斗描写

```markdown
❌ AI味：
西门吹雪的剑意凌厉无比，每一剑都蕴含着毁灭性的力量，敌人根本无法抵挡。

✅ 去AI味：
第三剑。
对手的刀断了。
西门吹雪收剑。
他看了一眼断刀，转身走了。
```

## 📝 写作检查清单（每章必查）

### AI味检查
- [ ] 无"光芒/情绪"类描写
- [ ] 无"完美/优雅/从容"形容
- [ ] 无过度心理描写
- [ ] 无"行云流水/仿佛"比喻

### 对话检查
- [ ] 简短有力（每句不超过20字）
- [ ] 有打断/沉默/不回应
- [ ] 口语化（像真人在说话）
- [ ] 无完整客套话

### 动作检查
- [ ] 动作具体可感
- [ ] 有小动作（摸鼻子、挠头等）
- [ ] 无"标准动作"描写

### 情感检查
- [ ] 克制（少用感叹号）
- [ ] 留白（不说透）
- [ ] 不完美（犹豫、失误）

## 🎭 角色AI味特点

### 凌午夜
```
❌ 避免：路飞式无敌乐观全程
✅ 应该：偶尔沉默、偶尔不确定
```

### 东方凌薇
```
❌ 避免：话痨全程都是有用信息
✅ 应该：话多但经常跑题、有废话
```

### 南宫墨
```
❌ 避免：全程冷漠无感情
✅ 应该：偶尔露出不耐烦、被逗时反应
```

### 北冥瑶
```
❌ 避免：全程腹黑微笑
✅ 应该：有时候真的在笑、有时候整人过头
```

### 中原星辰
```
❌ 避免：中二全程无敌
✅ 应该：有时候尴尬、有时候说错话
```

### 西门吹雪
```
❌ 避免：全程高冷
✅ 应该：被夸帅时真的生气（傲娇）
```

## 🔄 改稿流程

```
初稿 → 第一遍去AI味 → 第二遍角色检查 → 第三遍节奏调整 → 终稿
```

### 第一遍：去AI味（最多）
- 删除所有禁用词
- 心理描写改动作
- 对话精简

### 第二遍：角色检查
- 对照档案卡检查外貌/穿着
- 检查口头禅使用
- 检查性格一致性

### 第三遍：节奏调整
- 删除废话
- 增加留白
- 节奏慢的地方加快
