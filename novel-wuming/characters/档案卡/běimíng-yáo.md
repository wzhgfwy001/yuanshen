## 👤 外貌特征

### 身形
- 身高：162cm（入学）→ 165cm（21岁）
- 体型：纤细柔美，身材比例好，有锻炼痕迹但不明显
- 姿态：站姿优雅，坐姿经常歪着，给人慵懒感

### 脸部
- 脸型：小巧的鹅蛋脸，下巴尖
- 眼睛：杏眼，眼神温柔但偶尔狡黠，笑起来弯弯的
- 眉毛：弯弯的淡眉，不刻意描画
- 嘴唇：薄但笑起来很好看，抿嘴时有微妙弧度
- 特色：经常似笑非笑，让人猜不透

### 标志形象
- **平时**：黑长直发，通常中分或偏分
- **战斗**：阴阳师服饰狩衣，黑白为主
- **日常**：经常戴着各种小发夹，最爱狐狸发夹
- **标志笑容**：似笑非笑，嘴角弧度固定，让人不知道她在想什么

## 👔 穿着风格（全程保持一致）

### 日常穿着
```
标准配置：
- 龙腾学院校服（太玄学院款，款式略有不同）
- 校服习惯：袖子通常挽到手肘，裙摆经常不扣扣子
- 内搭：白色或浅色吊带背心
- 袜子：白色长筒袜（经常是不同品牌的同款）
- 鞋子：简约帆布鞋，最常穿白色
- 饰品：各种发夹，狐狸发夹是标志性饰品

不喜欢：太正式的衣服、正装、领带
```

### 战斗穿着
```
标准配置：
- 阴阳师服饰：狩衣形式，黑白配色
- 狩衣特点：简洁利落，方便行动
- 辅助配件：护腕、护腿（轻便型）
- 面具：召唤时会戴，但经常不戴
- 不穿暴露的服装（这一点与很多女角色不同）

饰品：银手链（旧的，看起来有故事）
```

### 禁止穿着
- ❌ 粉色（她说是"幼稚的颜色"）
- ❌ 过于华丽的服装
- ❌ 任何带卡通图案的

## 🎭 性格层次

### 表层性格
```
【表现】腹黑温柔
- 平常有点腹黑，喜欢整人
- 表面温柔，笑起来很甜
- "呵呵～"是她的招牌笑
- 看人出丑时会笑得更开心
- 但整人有个度，不会太过分
```

### 深层性格

#### 隐藏面 #1：自卑与恐惧
```
【触发章节】：200-300章
【表现】：
- 得知自己身世后（阴阳师血脉）
- 开始担心自己是"怪物"
- 担心凌午夜和伙伴会因此疏远她
- 这才是她腹黑的真正原因——用强势掩饰脆弱

【真相】：她极度害怕失去伙伴
```

#### 隐藏面 #2：权力渴望
```
【触发章节】：400-500章
【表现】：
- 作为阴阳师血脉，她对权力有本能的渴望
- 这个发现让她害怕
- 她不想变成"为了权力不择手段的人"

【笔匠评价】：这是真实的欲望，不需要否定
```

#### 隐藏面 #3：姐姐的负担
```
【触发章节】：600章后
【表现】：
- 作为役小角后裔，她背负着家族期待
- 但她不想当"家主"
- 想当普通女孩
- 这与她的命运产生冲突
```
