## 👤 外貌特征

### 身形
- 身高：175cm（入学16岁）→ 180cm（21岁）
- 体型：精瘦但不单薄，肌肉线条流畅，有锻炼痕迹
- 姿态：站姿随意但重心稳，不喜欢正襟危坐

### 脸部
- 脸型：偏瘦的瓜子脸，轮廓清晰
- 眼睛：左眼比右眼略深（伏笔F001），清澈的黑色，专注时会变成银色
- 眉毛：剑眉，不粗重但很有型，修剪过但不完全整齐
- 嘴唇：厚度适中，放松时微翘，紧张时抿成一条线
- 肤色：普通偏白，不刻意晒黑

### 标志形象
- **战斗形态**：瞳孔收缩变成银色，左眼周围浮现银色纹路，银光从眼角溢出
- **平时**：眼神清澈带笑意，看起来人畜无害
- **严肃时**：眉头紧锁，嘴角下压，判若两人

## 👔 穿着风格（1000章全程保持一致）

### 日常穿着
```
标准配置：
- 龙腾学院校服（蓝色为主）
- 校服习惯：经常敞开穿，最上面两颗扣子不系
- 内搭：黑色背心或灰色T恤，轮换着穿
- 下装：深色运动裤或牛仔裤，不穿正装西裤
- 鞋子：普通运动鞋（经常不系鞋带）

```

### 战斗穿着
```
标准配置：
- 黑色紧身战斗服（定制款，轻便）
- 护腕护膝（东方凌薇特别设计的纳米材料款）

- 鞋子：高帮战斗靴（偶尔穿）
- 不穿护甲（嫌重）
```

- ❌ 白色袜子（他自己说太亮）
- ❌ 任何正装（除非姑姑逼他）

## 🎭 性格

### 核心性格：阳光乐天派

```
【定位】：不是"成长"，是"绽放"

凌午夜的天真是他的力量，不是缺陷
他的乐观不是无知，是选择
```

### 性格表现

#### 前期（1-200章）：路飞式乐天派
- 天真无畏，"我要成为最强的！"
- 对危险没概念往前冲
- 绝对信任伙伴
- "我们是伙伴啊！"
- 打架是交朋友的方式
- 笑容灿烂，咧着嘴
- 爱凑热闹，爱打抱不平
- 不记仇

#### 中期（201-600章）：成长中
- 同伴受重伤，开始意识到"我还不够强"
- 但不会因此变冷漠，而是更努力
- 笑容依然在，只是少了点傻气
- "我一定要更强，才能保护大家"
- 战斗时不再莽撞，但乐观没变

#### 后期（601-1000章）：温暖坚定
- 沉稳内敛但不失幽默
- 决策果断但会听取意见
- 乐观勇敢探索精神一直都在
- 真正的领袖气质
- "不管我是谁，我选择当什么样的人"

## ⚡ 能力设定（1000章版）

### 时空操控（左眼）

| 阶段 | 章节 | 能力表现 |
|------|------|----------|
| 觉醒前期 | 1-10 | 左眼异常闪光，短暂预知 |
| 觉醒期 | 11-50 | 时空裂隙，范围小，不可控 |
| 成长期 | 51-200 | 时空凝固，可中断小范围时间流 |
| 稳定期 | 201-400 | 时空操控，可同时操作2-3个目标 |
| 精通期 | 401-600 | 时空转移，可带人短距离瞬移 |
| 大师期 | 601-800 | 时空回溯（有限），代价是消耗寿命 |
| 完全体 | 801-1000 | 完整时空操控，但有道德限制 |

**能力代价：**
- 每次使用大型时空技能，左眼会流血
- 使用过于强大的能力会导致头痛和暂时失明
- 时空回溯的代价是消耗生命力

**能力来源：** 父亲凌不凡的遗传 + 宝塔第三层觉醒

## 🌟 成长弧线（阳光成长型）

### 核心定位
```
凌午夜的天真是他的力量，不是缺陷
凌午夜的乐观不是无知，是选择
```

### 成长阶段

| 阶段 | 章节 | 成长内容 | 核心转变 |
|------|------|----------|----------|
| 起点 | 1 | 阳光开朗，笑容灿烂 | - |
| 第一次挑战 | 101 | 能力不足导致失误，但积极补救 | "我还不够强" |
| 第二次挑战 | 300 | 身世真相，面对选择 | "我选择当什么样的人" |
| 第三次挑战 | 500 | 领袖责任，两难处境 | "我不想做选择，但我必须" |
| 第四次挑战 | 700 | 承担代价，继续前行 | "这就是代价，我会承担" |
| 终点 | 1000 | 温暖坚定，守护伙伴 | 真正的领袖 |

### 成长特点
- **不是黑化，是坚强**：遇到挫折会沮丧，但不会变成冷酷的人
- **不是无敌，是进步**：能力会不足，但努力弥补
- **不是完美，是真实**：会犯错，但记得自己为什么而战
- **不是孤独，是担当**：承担领袖的责任，但不是孤独地战斗

## ⚠️ 成长挑战（替代灰度设定）

**核心原则：** 凌午夜的"不完美"是成长空间，不是暗黑倾向

### 挑战 #1：能力不足
```
【章节】：预计101-150章
【表现】：
第一次执行任务时，因为能力不足导致进度落后
这次经历让他更加努力训练

【结果】：更加努力，不是自我怀疑
```

### 挑战 #2：信息太多
```
【章节】：预计300-350章
【表现】：
获得了太多关于身世的信息，一时不知如何处理
但最终向伙伴坦白，一起面对

【结果】：学习如何处理复杂信息，不是隐瞒
```

### 挑战 #3：两难抉择
```
【章节】：500章后
【表现】：
作为领袖，必须在两个选项中选择
他会有挣扎，但最终选择保护伙伴

【结果】：承担选择的责任，但不是冷酷
```
