## 💫 能力

### 觉醒前（平时状态）

| 能力 | 说明 |
|------|------|
| **武力** | 普通人水平，比凌午夜还弱 |
| **学习** | 成绩平庸，被当"吊车尾" |
| **异能** | 无明显异能表现 |
| **特殊** | 心灵感应（与姐姐宁月初有微弱联结） |

### 觉醒后（S+级潜力）

| 能力 | 说明 |
|------|------|
| **核心能力** | 能力回流 + 共鸣放大 |
| **能力回流** | 被姐姐吸收的能力全部回归 |
| **共鸣效果** | 与宁月初能力共振放大 |
| **最终等级** | S+级（与宁月初并列） |

**觉醒条件：**
> 宁月初遇到生命危险时触发
> → 姐弟心灵感应联结爆发
> → 能力回流 + 共鸣放大
> → 双方同时提升至S+级

## 📖 故事线

### 前期（1-60章）
- 表现平庸，被同学当"废物"
- 知道自己拖累姐姐，内心愧疚
- 与凌午夜建立深厚兄弟情
- 暗中观察姐姐的动向

### 中期（61-90章）
- 姐姐失踪，被迫面对现实
- 在寻找姐姐的过程中逐渐觉醒
- 发现自己的真实身世

### 高光（91-120章）
- 姐姐宁月初被围困，生命垂危
- **觉醒爆发！**
- 能力回流+共鸣放大，姐弟联手击败敌人
- 展现S+级潜力

## ⚠️ 重要设定

### 能力被吸收的真相
```
出生时：宁千阳与宁月初是双胞胎
    ↓
宁月初在出生瞬间自发吸收弟弟的能力
    ↓
宁千阳变成"普通人"
    ↓
宁月初变得极为强大（全能系+五行）
    ↓
宁千阳一直在背后默默守护
```

### 心灵感应联结
- 姐弟之间存在微弱的心灵感应
- 平时几乎感觉不到
- 觉醒时完全同步
