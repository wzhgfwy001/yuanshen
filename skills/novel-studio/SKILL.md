# 小说创作助手 - Novel Studio

**版本：** v1.2.0

**类型：** 行业套件

**功能：** 通过5阶段引导，帮助用户构建完整小说

**适用类型：** 科幻、奇幻、言情、悬疑、都市等

---

## 核心架构

```
用户输入："写一个科幻小说"
    ↓
【小说创作助手】
    ↓
阶段1：世界观构建
├── 山寺三郎（世界观设计）
├── 波特卡斯·D·艾斯（元素分析）
├── 金点子系统（用户添加创意）← 嵌入
└── 时间线规划（初始时间线）← 嵌入
    ↓ 用户确认
阶段2：剧情架构
├── 奈良鹿丸（情节设计）
├── 时间线闭环检测← 嵌入
└── 金点子融入确认
    ↓ 用户确认
阶段3：人物塑造
├── 朽木白哉（人物设定）
├── 人物卡生成（身形/脸型/穿着）← 强化
└── 人物一致性检查← 嵌入
    ↓ 用户确认
阶段4：正文撰写
├── 山寺三郎（正文）
├── 风格调用（用户习惯/模仿作家）← 嵌入
├── AI味检测（≤20%）← 嵌入
└── 金点子融入确认
    ↓ 用户确认
阶段5：审查润色
├── 质量检查Agent
├── 时间线复查（伏笔回收）
├── 最终AI味检测
└── 用户评分
```

## 需要新增的Agent

### 🔴 P0 - AI味检测Agent

**职责：**
- 检测AI写作特征
- 提供去AI味建议
- 评分（≤20%合格）

**AI味特征库：**
```
- "首先...其次...最后" 句式
- 过度使用"非常"
- 过于完美的转折
- 缺乏口语化表达
- 机械的因果关系
- 公式化情感表达
```

**输出：**
```json
{
  "ai_score": 15,
  "detected_features": ["过度使用连接词", "情感描写公式化"],
  "suggestions": ["增加口语化表达", "打散句式结构"]
}
```

---

### 🟡 P1 - 作品拆解Agent

**职责：**
- 分析输入作品的结构
- 提取核心元素
- 生成改编模板

**分析维度：**
```
输入作品 → 系统分析
    ↓
1. 故事大纲提取
2. 剧情转折点梳理
3. 人物关系图谱
4. 叙事结构分析
5. 风格特征识别
    ↓
输出：可复用的改编模板
```

**输出：**
```json
{
  "analysis": {
    "title": "原作品名",
    "structure": {
      "setup": "建置部分",
      "development": "发展部分",
      "climax": "高潮",
      "resolution": "解决"
    },
    "plot_twists": [...],
    "characters": [...],
    "adaptable_elements": [...]
  }
}
```

---

### 🟡 P1 - 时间线管理Agent

**职责：**
- 创建完整时间线
- 检测闭环率
- 确保伏笔回收

**时间线模板：**
```json
{
  "timeline": [
    {
      "chapter": 1,
      "title": "章节标题",
      "time": "时间点",
      "location": "地点",
      "events": ["事件1", "事件2"],
      "foreshadowing": ["伏笔1", "伏笔2"],
      "resolution": ["回收伏笔"]
    }
  ]
}
```

**闭环检查：**
```
闭环率 = 已回收伏笔 / 总伏笔数 × 100%
    ↓
合格标准：≥80%
    ↓
如果<80%，返回阶段2重新调整
```

---

### 🟡 P1 - 人物一致性Agent

**职责：**
- 生成人物卡
- 检查描写一致性
- 确保外形/性格/穿着全程统一

**人物卡格式：**
```
┌─────────────────────────────────────┐
│ 姓名：张三                            │
│ 年龄：28岁                            │
│ 身形：178cm，健壮                     │
│ 脸型：国字脸，浓眉                   │
│ 发型：短发，略显凌乱                  │
│ 眼神：深邃，沉稳                     │
│ 穿着：深色休闲装，简约风格            │
│ 配饰：左手戴旧式手表                  │
│ 性格：沉稳，内敛                       │
│ 口癖：说话前会停顿一下               │
│ 习惯：思考时皱眉                      │
└─────────────────────────────────────┘
```

**一致性检查：**
```
每次人物出场描写 → 检查是否偏离人物卡
    ↓
如果偏离 → 标记 + 建议修改
    ↓
用户确认后才可偏离
```

## 用户金点子系统

### 功能说明
用户在创作过程中可以随时添加金点子（剧情创意、人物设定等），并指定出现在哪个时间线或场景。

### 使用方式
```
用户：我想加一个金点子
系统：请描述您的创意
用户：[描述创意]
系统：请选择出现位置：
       1. 第3章-星际探索
       2. 第5章-第一次接触
       3. 新增时间线
用户：1
系统：已添加到第3章
```

### 金点子格式
```json
{
  "idea": "外星生物是友善的",
  "type": "剧情转折",
  "target_timeline": "第3章",
  "target_scene": "第一次接触"
}
```

## 时间线系统

### 要求
- **完整性**：每个情节点必须能在时间线上找到位置
- **闭环性**：故事开头和结尾必须有逻辑呼应
- **可追溯性**：任何设定变更必须更新时间线

### 闭环检查
```
开头伏笔 → 中间发展 → 结尾回收
    ↓
每个伏笔必须有对应回收点
    ↓
质量检查时验证闭环率（≥80%）
```


```
    ↓
```

## 去AI味系统

### 检测标准

| AI味特征 | 检测方式 |
| -------- | -------- |
| "首先...其次...最后" | 句式检测 |
| 过度使用"非常" | 频率检测 |
| 过于完美的转折 | 逻辑检测 |
| 缺乏口语化表达 | 对话风格检测 |
| 机械的因果关系 | 连接词检测 |
| 公式化情感表达 | 情感描写密度 |

**标准：AI味比例≤20%才合格**

### 去味方法
```
检测到AI味 → 标记 → 修改建议
    ↓
1. 替换过度使用的连接词
2. 增加口语化表达
3. 加入生活细节
4. 打散过于整齐的句式
5. 加入人物独特的说话方式
```

## 风格学习系统

### 功能1：模仿作家风格
```
用户：我想模仿金庸的写作风格
    ↓
系统分析金庸风格特征：
- 招式描写：具体、有画面感
- 人物命名：有寓意
- 叙事节奏：紧凑
- 对话风格：简洁有力
    ↓
应用到当前小说
```

### 功能2：学习用户风格
```
系统持续记录：
- 用户常用的词汇
- 用户的叙事节奏
- 用户的描写偏好
- 用户的句式习惯
    ↓
定期更新用户画像
    ↓
越用越像用户自己写的
```

### 风格库
```
用户上传/粘贴喜欢的作品 → 系统分析风格特点
    ↓
风格特点存入"风格库"：
    - 叙事节奏（快/中/慢）
    - 句式特点（短句/长句/混合）
    - 描写密度（简洁/中等/详细）
    - 对话风格（口语/书面/文学）
    - 特色表达（作家独有用法）
    ↓
创作时可调用这些风格
```

### 风格应用
```
用户：我想用金庸风格写这段
    ↓
系统调用金庸风格库：
- 招式描写：有画面感，具体
- 人物命名：有寓意
- 对话风格：简洁有力
- 叙事节奏：紧凑
    ↓
应用到当前章节
```

## 作品分析系统

### 功能
能够分析并拆解其他作品，用于学习和改编。

### 改编流程
```
输入：某经典故事/用户粘贴文本
    ↓
系统拆解：
├── 核心主题（爱情/复仇/成长...）
├── 人物原型（英雄/智者/影子...）
├── 情节模板（起承转合结构）
├── 世界观元素
    ↓
用户选择：
├── 新题材（科幻/奇幻/现代...）
└── 改编程度（轻度/中度/完全重写）
    ↓
系统重组 → 新故事大纲
    ↓
用户确认 → 进入5阶段创作
```

---

## 工作流程

### 通用规则
- **每阶段完成后，必须用户确认**
- **用户确认后，才进入下一阶段**
- **如果不满意，返回该阶段重新调整**
- **金点子可随时插入**
- **人物卡一旦锁定，描写必须遵循**

### 交付物标准

| 阶段 | 交付物 | 说明 |
|------|--------|------|
| 阶段1 | 世界观文档 | 包含时间/空间/社会/科技/文化 |
| 阶段2 | 故事大纲+时间线 | 三幕结构+完整时间线（闭环） |
| 阶段3 | 人物设定表+人物卡 | 主角/反派/配角（含外貌/穿着/性格） |
| 阶段4 | 章节草稿 | 每章800-2000字，去AI味版 |
| 阶段5 | 定稿 | AI味≤20%，闭环≥80% |
