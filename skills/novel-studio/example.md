# 小说创作助手示例

## 示例：创作一部科幻小说

### 用户输入
```
我要写一个科幻小说，关于星际探索的
```

### 系统响应

```
🎬 小说创作助手已启动

请选择小说类型：
1. 科幻
2. 奇幻
3. 言情
4. 悬疑
5. 都市
```

### 用户选择：1（科幻）

---

### 阶段1：世界观构建

**山寺三郎** 正在设计世界观...

**波特卡斯·D·艾斯** 正在分析科幻元素...

```
🌌 世界观草案

时间背景：公元2150年
空间背景：银河系边缘的"曙光星系"
社会形态：星际联邦制，各星球自治
科技水平：曲速引擎、量子传送门
文化特征：探索精神、星际贸易

是否确认？回复"确认"或"修改[具体内容]"
```

### 用户：确认

---

### 阶段2：剧情架构

**奈良鹿丸** 正在设计剧情...

```
📖 剧情架构草案

核心主题：人类对未知的探索精神
主要冲突：联邦与边缘星球的资源争夺

三幕结构：
第一幕（1-10章）：发现曙光星系
第二幕（11-25章）：与外星文明接触
第三幕（26-30章）：星际战争与和平

是否确认？回复"确认"或"修改"
```

### 用户：确认

---

### 阶段3：人物塑造

**朽木白哉** 正在设计角色...

```
👥 角色设定草案

【主角】林远航
- 身份：联邦探索舰队"曙光号"舰长
- 性格：勇敢、理性、有责任感
- 目标：发现新家园
- 缺陷：有时过于自信

【反派】卡尔·维森
- 身份：边缘星球联盟领袖
- 性格：狡猾、野心勃勃
- 目标：独立统治曙光星系

是否确认？回复"确认"或"修改"
```

### 用户：确认

---

### 阶段4：正文撰写

**山寺三郎** 正在撰写正文...

```
📝 正在创作中...

第一章：曙光
（约1500字）

[正文内容省略...]
```

### 用户：继续

---

### 阶段5：审查润色

**质量检查Agent** 正在审查...

```
✅ 审查完成

逻辑连贯性：✅ 通过
人物一致性：✅ 通过
情节合理性：✅ 通过
语言流畅度：✅ 通过
错别字检查：✅ 通过

综合评分：⭐⭐⭐⭐⭐（92分）
```

---

### 最终输出

```
🎉 小说创作完成！

《星际曙光》
类型：科幻
字数：约45000字（30章）

结构：
├── 第一章：曙光
├── 第二章：启航
├── 第三章：曲速
...（共30章）

文件已保存至：~/novels/stellar-dawn/
```

---

## 使用场景

| 用户需求 | 使用指令 |
|----------|----------|
| 从零创作 | "写小说" |
| 继续写作 | "继续写作" |
| 基于大纲 | "帮我写[具体章节]" |
| 风格调整 | "改成悬疑风格" |
