# Lossless Claw

## 🎯 快速使用

**触发场景：**
- 用户说"整理会话记录"、"蒸馏记忆" → 自动调用此Skill
- 用户说"提取偏好"、"总结任务" → 自动调用此Skill
- 用户说"长期记忆"、"无损回忆" → 自动调用此Skill
- 用户说"会话摘要"、"提取知识" → 自动调用此Skill

**示例对话：**
- 用户："帮我整理一下最近的会话记录"
- AI自动使用 `distill` 命令处理，提取身份、偏好、任务、知识

- 用户："把今天的对话蒸馏成长期记忆"
- AI自动执行蒸馏流程，保留重要内容到长期层

- 用户："提取用户的偏好设置"
- AI自动从会话中提取偏好信息并结构化存储

**不适用场景：**
- 简单问答（无实质内容可提取）不使用此Skill
- 闲聊（纯对话无知识沉淀）不使用此Skill
- 需要实时处理当前对话时不使用此Skill（仅处理已完成会话）

## 工作流

通过"短期全量记录 + 长期结构化沉淀"实现无损回忆。

1. 读取会话文本或日志
2. 提取身份、偏好、任务、知识
3. 将原始内容保留到短期日志
4. 将高价值内容提升到长期层
5. 生成可追溯摘要

## 推荐命令

```bash
python3 scripts/personal_ai_memory.py distill --input /path/to/session.md
python3 scripts/personal_ai_memory.py distill --input /path/to/session.md --apply
```

## 输出目标

- 重要内容永久固化
- 无效噪声不进入长期层
- 保留完整短期回溯链路
