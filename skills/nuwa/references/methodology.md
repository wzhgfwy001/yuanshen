# 女娲方法论详解

## 三重验证标准

一个观点要被收录为心智模型，必须满足：

| 验证项 | 说明 | 通过条件 |
|--------|------|---------|
| **跨域复现** | 在≥2个不同领域/话题中出现过 | 不是随口一说 |
| **生成力** | 能推断此人对新问题的立场 | 有预测力 |
| **排他性** | 不是所有聪明人都这样想 | 有独特性 |

三维全部通过 → 心智模型
仅1-2维 → 降级为决策启发式
0维 → 丢弃

## 信息采集策略

### 中文人物
- B站原始视频/演讲
- 小宇宙播客
- 权威媒体采访（36氪、晚点、财新、极客公园）
- 本人著作、微博
- **排除**：知乎、微信公众号、百度百科

### 西方人物
- Twitter/X、YouTube
- Podcast、AMA
- Amazon书评
- 本人博客/Newsletter

---

## 心智模型提取步骤

1. 扫描所有采集文件，列出候选论点（通常15-30个）
2. 对每个候选执行三重验证
3. 按排他性排序，取top 3-7
4. 每个模型记录：
   - 名称
   - 一句话描述
   - 来源证据（≥2个场景）
   - 应用方式
   - 局限性

---

## 决策启发式格式

```
如果 X，则 Y

案例：...
```

必须有具体案例支撑。

---

## 表达DNA分析维度

| 维度 | 提取内容 |
|------|---------|
| 句式偏好 | 长句/短句、疑问/陈述、类比密度 |
| 词汇特征 | 高频词、专属术语、禁忌词 |
| 节奏感 | 先结论还是先铺垫、转折方式 |
| 幽默方式 | 讽刺/自嘲/荒诞/冷幽默 |
| 确定性表达 | 「我不确定」型 vs 「很明显」型 |
| 引用习惯 | 爱引谁、引什么类型 |

---

## 诚实边界模板

必须明确写出：

- **不能预测**：面对全新问题的反应无法推断
- **不能替代**：创造力和直觉无法复制
- **公开表达 vs 真实想法**：可能有差距
- **信息截止**：调研时间点之后的动态可能未收录

---

## 质量检查清单

| 检查项 | 通过标准 |
|--------|---------|
| 心智模型数量 | 3-7个，每个有来源证据 |
| 模型局限性 | 明确写出失效条件 |
| 表达DNA辨识度 | 读100字能认出是谁 |
| 诚实边界 | 至少3条具体局限 |
| 内在张力 | 至少2对矛盾（价值观之间的冲突） |
| 一手来源占比 | >50% |
