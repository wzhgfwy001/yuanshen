## 🎯 常见问题回复模板

### Q1: Token 优化效果如何？
**回复：** v1.1.0 版本通过 8 大优化模块，Token 消耗降低 90%，成本降低 95%，速度提升 3-8 倍。实际效果因任务类型而异，欢迎分享您的使用数据！

### Q2: 支持哪些任务类型？
**回复：** 系统支持 4 种任务类型：
- 单一任务（简单查询/计算）
- 标准任务（需要 2-3 步处理）
- 创新任务（创作/分析/规划）
- 复杂任务（多步骤、多角色协作）

### Q3: 如何查看任务执行状态？
**回复：** 使用内置监控大屏，访问 `http://localhost:5000/dashboard-simple.html` 即可实时查看任务队列和 Agent 状态。

### Q4: 遇到问题如何反馈？
**回复：** 欢迎在评论区留言，或提供以下信息：
1. 任务类型和描述
2. 错误信息（如有）
3. 期望的输出
我们会尽快响应！
