# 📚 学术论文写作示例

**示例类型：** 学术论文  
**难度：** 高  
**预计耗时：** 90-120 分钟

---

## 📝 任务描述

```
请使用学术论文模板，撰写一篇关于"深度学习在医学影像分析中的应用"的学术论文。

要求：
- 标题：Deep Learning in Medical Image Analysis: A Comprehensive Survey
- 摘要：200-300 字，包含研究目的、方法、结果、结论
- 关键词：5 个，包括 deep learning, medical image analysis 等
- 学科领域：人工智能/医学
- 目标字数：5000 字
- 引用格式：IEEE

章节结构：
1. 摘要
2. 引言（研究背景、问题陈述、研究意义）
3. 相关工作（深度学习基础、医学影像分析现状）
4. 方法（数据收集、模型架构、训练策略）
5. 结果（实验设置、对比结果、消融实验）
6. 讨论（结果分析、局限性、未来方向）
7. 结论
8. 参考文献
```

---

## 🎯 任务分解

### 子任务 1：文献检索与整理
**Agent 角色：** 文献综述师  
**任务描述：** 搜集深度学习在医学影像分析领域的相关文献  
**输出：** 文献列表（20-30 篇），按主题分类  
**预计耗时：** 20 分钟

### 子任务 2：摘要与引言撰写
**Agent 角色：** 学术顾问  
**任务描述：** 撰写摘要和引言部分  
**输出：** 摘要（300 字）+ 引言（800 字）  
**预计耗时：** 15 分钟

### 子任务 3：相关工作撰写
**Agent 角色：** 文献综述师  
**任务描述：** 综述深度学习和医学影像分析的研究现状  
**输出：** 相关工作章节（1000 字）  
**预计耗时：** 20 分钟

### 子任务 4：方法章节撰写
**Agent 角色：** 数据分析师  
**任务描述：** 描述数据收集、模型架构、训练策略  
**输出：** 方法章节（1500 字）  
**预计耗时：** 25 分钟

### 子任务 5：结果与讨论撰写
**Agent 角色：** 数据分析师 + 学术顾问  
**任务描述：** 呈现实验结果，分析讨论  
**输出：** 结果（1000 字）+ 讨论（800 字）  
**预计耗时：** 25 分钟

### 子任务 6：结论与参考文献整理
**Agent 角色：** 文字编辑  
**任务描述：** 撰写结论，整理参考文献  
**输出：** 结论（400 字）+ 参考文献列表  
**预计耗时：** 15 分钟

### 子任务 7：全文审查与润色
**Agent 角色：** 审查专家  
**任务描述：** 全面审查论文质量，润色文字  
**输出：** 最终版本论文  
**预计耗时：** 20 分钟

---

## 👥 Agent 团队配置

| 角色 | 数量 | 职责 |
|------|------|------|
| 学术顾问 | 1 | 确保论文结构完整、逻辑严谨 |
| 文献综述师 | 1 | 搜集整理文献，确保引用完整 |
| 数据分析师 | 1 | 分析实验数据，生成图表 |
| 文字编辑 | 1 | 润色文字，确保语言流畅 |
| 审查专家 | 1 | 全面审查论文质量 |

**总计：** 5 个子 Agent

---

## 📊 执行流程

```
[1] 文献检索 → [2] 摘要引言 → [3] 相关工作
                          ↓
                    [4] 方法章节
                          ↓
                  [5] 结果与讨论
                          ↓
                  [6] 结论与参考文献
                          ↓
                    [7] 审查润色
                          ↓
                      [完成]
```

## ✅ 质量检查

### 结构完整性
- [ ] 所有章节是否完整？
- [ ] 摘要是否包含四要素？
- [ ] 引言是否明确问题？
- [ ] 方法是否详细？
- [ ] 结果是否有数据？
- [ ] 讨论是否深入？
- [ ] 结论是否回应问题？

### 学术规范
- [ ] 引用是否准确？
- [ ] 格式是否符合 IEEE？
- [ ] 是否有抄袭嫌疑？
- [ ] 数据是否真实？
- [ ] 图表是否清晰？

### 语言表达
- [ ] 语句是否通顺？
- [ ] 用词是否准确？
- [ ] 是否学术化表达？
- [ ] 逻辑是否连贯？
- [ ] 是否有语法错误？
