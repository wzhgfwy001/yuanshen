# 🔧 代码重构示例

**示例类型：** 代码重构  
**难度：** 中  
**预计耗时：** 60-90 分钟

---

## 📝 任务描述

```
请对以下遗留代码进行重构，提升代码质量和可维护性。

原始代码问题：
- 函数过长（>200 行）
- 重复代码多
- 命名不规范
- 缺乏注释
- 无单元测试
- 耦合度高

重构目标：
- 函数长度 <50 行
- 消除重复代码
- 规范命名
- 添加充分注释
- 单元测试覆盖率 >80%
- 降低耦合度

重构范围：
1. 代码结构优化
2. 命名规范化
3. 提取公共方法
4. 添加注释文档
5. 编写单元测试
6. 性能优化
```

---

## 🎯 任务分解

### 子任务 1：代码分析与问题识别
**Agent 角色：** 代码审查师  
**任务描述：** 分析代码质量问题，列出重构清单  
**输出：** 代码分析报告，包含问题列表、优先级  
**预计耗时：** 15 分钟

### 子任务 2：代码结构重构
**Agent 角色：** 重构专家  
**任务描述：** 拆分大函数，提取公共方法，消除重复代码  
**输出：** 重构后的代码，函数长度<50 行  
**预计耗时：** 25 分钟

### 子任务 3：命名规范化
**Agent 角色：** 代码规范师  
**任务描述：** 重命名变量、函数、类，使其达意  
**输出：** 命名规范的代码  
**预计耗时：** 10 分钟

### 子任务 4：注释与文档
**Agent 角色：** 文档工程师  
**任务描述：** 添加函数注释、复杂逻辑说明  
**输出：** 注释充分的代码  
**预计耗时：** 15 分钟

### 子任务 5：单元测试编写
**Agent 角色：** 测试工程师  
**任务描述：** 编写单元测试，确保覆盖率>80%  
**输出：** 单元测试文件，覆盖率报告  
**预计耗时：** 25 分钟

### 子任务 6：性能优化与验证
**Agent 角色：** 性能工程师  
**任务描述：** 优化性能瓶颈，验证功能正确性  
**输出：** 性能优化报告，测试通过报告  
**预计耗时：** 15 分钟

---

## 👥 Agent 团队配置

| 角色 | 数量 | 职责 |
|------|------|------|
| 代码审查师 | 1 | 分析代码质量问题 |
| 重构专家 | 1 | 执行代码重构 |
| 代码规范师 | 1 | 规范命名和格式 |
| 文档工程师 | 1 | 添加注释文档 |
| 测试工程师 | 1 | 编写单元测试 |
| 性能工程师 | 1 | 优化性能 |

**总计：** 6 个子 Agent

## 📊 执行流程

```
[1] 代码分析 → [2] 结构重构 → [3] 命名规范
                              ↓
                        [4] 注释文档
                              ↓
                        [5] 单元测试
                              ↓
                    [6] 性能优化
                              ↓
                    [验证通过]
                              ↓
                        [完成]
```
