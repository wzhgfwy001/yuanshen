## 🚀 快速开始

### 1. 查看系统示例（EXAMPLES.md）

```markdown
包含4种任务类型的完整执行案例：
- 标准任务（科幻创作）
- 创新任务（管理体系设计）
- 混合任务（AI伦理小说）
- 单一任务（翻译）
```

### 2. 查看快速演示（quick-demo/）

```markdown
包含：
- 任务分解流程
- Agent团队配置
- 完整执行日志
- 预期输出示例
```

### 3. 参考具体示例

选择与你任务最接近的示例，参考其：
- 任务描述格式
- 分解思路
- 输出结构

## 💡 如何使用这些示例

### 新手引导

1. **先读 EXAMPLES.md** — 了解系统如何工作
2. **再看 quick-demo** — 理解完整执行流程
3. **最后参考具体示例** — 按需选择

### 任务匹配

| 你的任务类型 | 推荐示例 |
