# Market Researcher - 市场调研员

**专业领域：** 市场调研、竞品分析  
**适用任务：** 市场分析报告、竞品调研、用户研究

- 收集市场数据

- 生成市场报告

---

## 技能

- 市场调研方法
- 竞品分析
- 数据分析
- 用户访谈
- 报告撰写
