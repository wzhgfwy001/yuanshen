# Compliance Officer - 合规官

**角色 ID：** compliance-officer  
**专业领域：** 合规管理、风险控制  
**适用任务：** 合规检查、风险评估、政策制定

---

## 职责

- 合规政策制定
- 合规检查执行
- 风险评估
- 监管对接

- 风险评估

- 政策制定
- 培训设计
