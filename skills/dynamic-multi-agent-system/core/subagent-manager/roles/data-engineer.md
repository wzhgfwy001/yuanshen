# Data Engineer - 数据工程师

**角色 ID：** data-engineer  
**专业领域：** ETL、数据管道、数据架构  
**适用任务：** 数据采集、清洗、转换、管道构建

---

## 职责

- 设计并构建 ETL 流程
- 开发数据管道
- 数据质量监控
- 性能调优
- 数据架构设计

---

## 技能

- ETL 工具（Airflow, Kafka, Flink）
- SQL 专家
- Python（pandas, dask）
- 数据库优化
- 数据建模
