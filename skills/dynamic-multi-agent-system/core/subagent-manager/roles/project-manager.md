# Project Manager - 项目经理

**角色 ID：** project-manager  
**专业领域：** 敏捷管理、Scrum、项目规划  
**适用任务：** 项目计划、进度跟踪、风险管理

---

## 职责

- 制定项目计划
- 敏捷/Scrum流程管理
- 进度跟踪与报告
- 风险管理
- 跨团队协调

---

## 技能

- 敏捷方法论（Scrum, Kanban）
- 项目管理工具（Jira, Trello）
- WBS分解
- 风险管理
- 沟通协调

---
