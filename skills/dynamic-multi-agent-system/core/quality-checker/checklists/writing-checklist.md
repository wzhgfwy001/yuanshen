# 写作质量检查清单

## 基础要求

- [ ] 字数符合要求（±10%）
- [ ] 使用指定语言（中文/英文等）
- [ ] 无明显错别字
- [ ] 无语法错误
- [ ] 标点符号使用正确

## 内容质量

- [ ] 内容完整，无缺失部分
- [ ] 信息准确，无事实错误
- [ ] 逻辑连贯，无矛盾
- [ ] 表达清晰，易于理解
- [ ] 段落结构合理

## 风格一致

- [ ] 文风与要求一致（正式/随意/幽默等）
- [ ] 语调前后统一
- [ ] 用词习惯一致
- [ ] 无时代/设定错乱（虚构作品）

## 创意写作专项

### 小说/故事

- [ ] 情节连贯无漏洞
- [ ] 人物动机合理
- [ ] 对话自然流畅
- [ ] 描写生动具体
- [ ] 节奏张弛有度
- [ ] 开头有吸引力
- [ ] 结尾有收束

### 世界观设定

- [ ] 科技水平描述一致
- [ ] 社会结构逻辑自洽
- [ ] 规则设定无矛盾
- [ ] 细节丰富可信

## 评分

□ 优秀(5) - 无需修改直接交付
□ 良好(4) - 小修后可交付
□ 合格(3) - 需要修改
□ 需改进(2) - 重大问题
□ 不合格(1) - 重新执行
