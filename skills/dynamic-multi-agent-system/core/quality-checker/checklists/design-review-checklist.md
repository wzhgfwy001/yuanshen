# 🎨 设计方案检查清单

**版本：** v1.0  
**适用场景：** 设计方案审查/UIUX 设计  
**检查阶段：** 设计评审/交付前

---

## 🎯 设计目标检查

### 需求对齐
- [ ] 是否理解业务目标？
- [ ] 是否满足用户需求？
- [ ] 是否符合产品定位？
- [ ] 是否解决核心问题？
- [ ] 是否有明确的成功指标？

### 用户研究
- [ ] 是否有用户画像？
- [ ] 是否有用户场景？
- [ ] 是否有用户旅程图？
- [ ] 是否有痛点分析？
- [ ] 是否有竞品分析？

### 设计原则
- [ ] 是否遵循设计原则？
- [ ] 是否保持一致性？
- [ ] 是否考虑可访问性？
- [ ] 是否考虑国际化？
- [ ] 是否考虑品牌调性？

## 🖼️ 视觉设计检查

### 布局设计
- [ ] 布局是否清晰？
- [ ] 信息层级是否明确？
- [ ] 视觉焦点是否突出？
- [ ] 留白是否合理？
- [ ] 栅格是否使用？

### 色彩设计
- [ ] 配色是否和谐？
- [ ] 是否使用品牌色？
- [ ] 对比度是否足够？
- [ ] 色彩是否有含义？
- [ ] 是否考虑色盲用户？

### 字体设计
- [ ] 字体是否易读？
- [ ] 字号是否合适？
- [ ] 字重是否有层次？
- [ ] 行距是否合理？
- [ ] 字体数量是否控制（<3 种）？

### 图标与图片
- [ ] 图标风格是否统一？
- [ ] 图标是否表意清晰？
- [ ] 图片质量是否高？
- [ ] 图片是否相关？
- [ ] 版权是否清晰？

## 📱 交互设计检查

### 导航设计
- [ ] 导航是否清晰？
- [ ] 当前位置是否明确？
- [ ] 返回是否方便？
- [ ] 搜索是否易用？
- [ ] 面包屑是否需要？

### 操作流程
- [ ] 流程是否顺畅？
- [ ] 步骤是否最少？
- [ ] 反馈是否及时？
- [ ] 错误是否预防？
- [ ] 撤销是否支持？

### 表单设计
- [ ] 字段是否必要？
- [ ] 标签是否清晰？
- [ ] 占位符是否有帮助？
- [ ] 验证是否友好？
- [ ] 错误提示是否明确？

### 动效设计
- [ ] 动效是否有目的？
- [ ] 时长是否合适（200-500ms）？
- [ ] 缓动是否自然？
- [ ] 性能是否考虑？
- [ ] 是否可关闭？

## ♿ 可访问性检查

### 视觉可访问性
- [ ] 对比度是否达标（WCAG AA）？

- [ ] 颜色是否是唯一信息载体？
- [ ] 焦点是否可见？
- [ ] 动画是否可暂停？

### 操作可访问性
- [ ] 是否支持键盘操作？
- [ ] 点击区域是否够大（44x44px）？
- [ ] 是否有足够时间？
- [ ] 是否支持语音控制？
- [ ] 是否支持辅助技术？

### 内容可访问性
- [ ] 是否有 alt 文本？
- [ ] 是否有标题结构？
- [ ] 是否有语言属性？
- [ ] 是否有跳过链接？
- [ ] 是否有字幕？
