# 📚 学术论文检查清单

**版本：** v1.0  
**适用场景：** 学术论文写作  
**检查阶段：** 最终审查

---

## 📋 结构完整性检查

### 摘要（Abstract）
- [ ] 是否包含研究目的？
- [ ] 是否说明研究方法？
- [ ] 是否呈现实验结果？
- [ ] 是否给出主要结论？
- [ ] 字数是否在 200-300 字？
- [ ] 关键词是否 3-5 个？

### 引言（Introduction）
- [ ] 是否介绍研究背景？
- [ ] 是否明确研究问题？
- [ ] 是否说明研究意义？
- [ ] 是否概述论文结构？
- [ ] 是否引用关键文献？

### 相关工作（Related Work）
- [ ] 是否覆盖重要文献？
- [ ] 是否分类整理？
- [ ] 是否指出研究空白？
- [ ] 是否与本文工作对比？
- [ ] 引用格式是否正确？

### 方法（Methodology）
- [ ] 方法描述是否详细？
- [ ] 是否可重复实现？
- [ ] 是否说明实验设计？
- [ ] 是否描述数据来源？
- [ ] 是否说明评估指标？

### 结果（Results）
- [ ] 数据是否准确？
- [ ] 图表是否清晰？
- [ ] 是否有统计显著性？
- [ ] 是否对比基线方法？
- [ ] 是否分析结果原因？

### 讨论（Discussion）
- [ ] 是否解释结果含义？
- [ ] 是否对比相关研究？
- [ ] 是否说明创新性？
- [ ] 是否讨论局限性？
- [ ] 是否提出未来方向？

### 结论（Conclusion）
- [ ] 是否总结主要发现？
- [ ] 是否回应研究问题？
- [ ] 是否说明贡献？
- [ ] 是否简洁明了？
- [ ] 是否避免新信息？

## ✍️ 写作质量检查

### 语言表达
- [ ] 语句是否通顺？
- [ ] 用词是否准确？
- [ ] 是否避免口语化？
- [ ] 是否使用学术用语？
- [ ] 时态是否一致？

### 逻辑连贯
- [ ] 段落间是否连贯？
- [ ] 章节间是否衔接？
- [ ] 论证是否严密？
- [ ] 是否有逻辑跳跃？
- [ ] 是否有矛盾之处？

### 格式规范
- [ ] 标题层级是否正确？
- [ ] 图表编号是否连续？
- [ ] 公式格式是否规范？
- [ ] 参考文献格式是否统一？
- [ ] 页码是否连续？

---

## 🔍 技术细节检查

### 数据与实验
- [ ] 数据集是否描述清楚？
- [ ] 实验设置是否详细？
- [ ] 参数设置是否合理？
- [ ] 对比方法是否公平？
- [ ] 结果是否可验证？

### 图表质量
- [ ] 图表是否清晰？
- [ ] 图例是否完整？
- [ ] 坐标轴标签是否正确？
- [ ] 单位是否标注？
- [ ] 图表是否自包含？

### 引用与参考文献
- [ ] 引用是否准确？
- [ ] 是否引用最新文献？
- [ ] 是否避免过度引用？
- [ ] 是否避免遗漏重要文献？
- [ ] 格式是否符合要求？

## ⚠️ 常见问题检查

### 学术诚信
- [ ] 是否有抄袭嫌疑？
- [ ] 是否正确引用？
- [ ] 数据是否真实？
- [ ] 是否一稿多投？
- [ ] 作者贡献是否明确？

### 伦理合规
- [ ] 是否涉及人类受试者？
- [ ] 是否需要伦理审查？
- [ ] 数据使用是否合规？
- [ ] 是否存在利益冲突？
- [ ] 是否声明资助来源？
