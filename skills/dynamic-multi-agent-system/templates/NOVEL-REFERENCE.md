## §1 世界观核心设定（不可更改）

### 宇宙起源
```
大爆炸 → 外围生命先产生 → 能量耗尽 → 向中心逃离
                                        ↓
                            地球（能量补充站）
                                        ↓
                            九层宝塔（飞船/收集器）
                                        ↓
                            收集灵魂能量维持飞船动力
```

### 灵魂能量本质
```
灵魂能量 = 宇宙顶级能量形式

能力表现：
├── 影响空间 → 瞬移/空间门/空间折叠
├── 影响时间 → 回溯（理论）
├── 重塑物质 → 炼金/变形/元素操控
└── 改变因果 → 规则系能力

等级：F → E → D → C → B → A → S → SS → SS9
```

### 生命与灵魂能量关系
```
原始生命 → 灵魂能量极薄弱
    ↓进化
植物 → 基础能量（荣枯）
    ↓进化
动物 → 中等能量（轮回）
    ↓进化
人类 → 最精纯能量（智慧）

结论：越开智 = 能量越精纯
```

## §6 力量体系规则

### 灵魂能量构成
```
天魂（0.3） → 记忆/意识
地魂（0.3） → 情感/羁绊
人魂（0.4） → 肉体/根基

完整灵魂 = 天魂 + 地魂 + 人魂
遗民 = 只有天魂（纯灵魂体）
```

### 能力觉醒条件
```
普通人 → 觉醒灵魂能量 → 获得异能
        ↓
威胁/生死危机 → 触发更深度觉醒（姐弟共鸣）
```

### 吞噬者特性
```
本质：外来生物，非灵魂
能力：寄生/夺舍灵魂
弱点：怕纯能量攻击
目标：收集灵魂能量
```

## §8 禁止事项（绝对不能写）

### 世界观禁止
```
❌ 宝塔是地球文明起源
❌ 吞噬者本体已完全揭示
❌ 宇宙入侵完整真相
❌ 凌午夜有SS级以上能力（目前只揭示S级）
```

### 人物禁止
```
❌ 凌午夜称呼东方凌薇为"东方"
❌ 东方凌薇叫南宫墨"墨"
❌ 任何人叫凌午夜"少爷"
❌ 势力关系出现矛盾（如：教廷知道宝塔真相）
```

### 剧情禁止
```
❌ 凌午夜在第一部就完全掌控时空能力
❌ 姐弟共鸣在第10章前触发
❌ 吞噬者本体在第一部被击杀
```

## 🔄 锚定执行流程

```
1. 创作任务进入
   ↓
2. Agent读取本文档（强制）
   ↓
3. world-builder 检查世界观一致性
   ↓
4. character-editor 检查称呼规范
   ↓
5. content-writer 按锚定内容创作
   ↓
6. consistency-checker 二次检查
   ↓
7. 通过 → 输出
   未通过 → 返回修改
```
