# 视频模板 - 短视频脚本创作

**模板 ID：** video-short-script  
**类型：** 视频  
**适用场景：** 抖音/快手短视频、宣传片、教程视频

---

## 任务描述

创作短视频脚本

---

## Agent 角色配置

| 顺序 | 角色 | 职责 |
|------|------|------|
| 1 | video-script-writer | 创作脚本 |
| 2 | social-media-specialist | 优化传播性 |
| 3 | 内容创作者 | 完善台词 |
| 4 | test-engineer | 审核逻辑 |

---

## 输出要求

- 时长：30-60 秒
- 格式：分镜表格
- 包含：画面、台词、时长、备注

---

## Prompt 模板

```
请创作一个短视频脚本：

**主题：** [用户指定]
**时长：** 60 秒
**平台：** 抖音
**风格：** 有趣/科普/情感（可选）

**要求：**
1. 开场 3 秒抓住注意力
2. 内容节奏紧凑
3. 有记忆点/金句
4. 结尾有行动号召

**输出格式：**
| 时间 | 画面 | 台词 | 备注 |
|------|------|------|------|
| 0-3s | ... | ... | 开场 hook |
```
