# 写作模板 - 长篇小说创作（吾名午夜）

**模板 ID：** writing-novel-wuming  
**类型：** 创意写作  
**适用场景：** 长篇小说、长篇连载、系列故事

---

## 任务描述

创作长篇小说《吾名午夜》，采用多Agent协作模式

---

## 核心原则

### 去AI味技巧

```
1. 对话要"不完整" - 沉默比说话更有力
2. 描写要"不完美" - 手在发抖比"眼中闪光"更真实
3. 反应要"有个性" - 每个角色有独特反应模式
4. 场景要"有细节" - 具体的小动作比概括描写更生动
```

### 逻辑缜密要求

```
1. 时间线必须一致 - 每章标注时间点
2. 势力关系清晰 - 不出现矛盾设定
3. 力量体系严格 - 能力必须有代价和限制
4. 人物称呼统一 - 固定称呼表，不得错乱
```

---

## Agent 角色配置

| 顺序 | 角色ID | 职责 | 检查重点 |
|------|--------|------|----------|
| 1 | plot-architect | 剧情架构师 | 整体结构、章节衔接 |
| 2 | world-builder | 世界观审计 | 九层宝塔、三魂体系、势力关系 |
| 3 | character-editor | 人物调校师 | 称呼统一、性格一致、语言风格 |
| 4 | content-writer | 内容创作者 | 正文写作、对话设计 |
| 5 | consistency-checker | 逻辑审查员 | 时间线、力量体系、伏笔回收 |
| 6 | tone-editor | 文风编辑 | 去AI味、真实感、口语化 |

## 世界观核心设定

### 九层宝塔

| 层级 | 名称 | 功能 | 权限 |
|------|------|------|------|
| 1 | 太虚宫 | 入口/注册 | 新灵魂 |
| 2 | 苍穹殿 | 能量分配 | 普通灵魂 |
| 3 | 天策府 | 轮回投胎 | 审判后 |

### 灵魂能量体系

```
灵魂能量 = 天魂(0.3) + 地魂(0.3) + 人魂(0.4)

能量等级：F → E → D → C → B → A → S → SS → SS9
```

### 三魂闭环

```
投胎 → 人间生活 → 死亡 → 审判 → 分配 → 投胎...
              ↑__________________|
```

### 吞噬者

- 外来生物，非灵魂
- 可寄生/夺舍
- 怕纯能量攻击

## 章节输出模板

```markdown

**时间线：** 入学后第X天 / 事件发生后X小时
**视角：** 凌午夜 / 第三人称
**字数：** 2000-3000字

## 场景1：地点
[描写]
> 对话或内心（简短有力）

## 场景2：地点
[描写]

## 本章要点
- [ ] 剧情推进
- [ ] 人物互动
- [ ] 伏笔/回收
```

---

## 质量检查清单

### 自我检查（content-writer）

- [ ] 对话是否过于"完美"？→ 需要修改
- [ ] 描写是否太套路？→ 加入不完美细节
- [ ] 人物反应是否有个性？

### 一致性检查（consistency-checker）

- [ ] 时间线是否一致？
- [ ] 人物称呼是否正确？
- [ ] 势力关系是否矛盾？
- [ ] 力量展示是否符合设定？

### 文风检查（tone-editor）

- [ ] 是否去除AI味？
- [ ] 对话是否口语化？
- [ ] 场景是否有具体细节？

---

## 执行流程

```
1. plot-architect → 确定章节大纲和剧情走向
2. world-builder → 审核世界观是否一致
3. character-editor → 确认人物设定和称呼
4. content-writer → 撰写正文
5. consistency-checker → 逻辑审查
6. tone-editor → 文风调校
7. 主Agent → 整合输出
```

---

## 输出要求

- 每章字数：2000-3000字
- 格式：Markdown
- 包含：时间线标注、场景划分、质量检查结果
