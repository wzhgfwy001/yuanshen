# 标准任务输出模板

## 适用场景

- 类型2：标准任务 (Standard Task)
- 类型3：创新任务+标准任务 (Hybrid Task)
- 诊断结果：L0 - 执行明确
- 复杂度：中

## 输出结构

```
## 📋 执行摘要
[1-2句话概括完成的工作]

---

## 📌 关键信息
[用户最关心的3-5个要点]

---

## 📄 详细内容
[完整输出内容]

---

## 💡 建议/后续步骤
[可选：下一步行动建议]
```

## 各部分说明

### 执行摘要

```
格式：动词 + 宾语 + 结果
示例：完成了科幻小说第一章创作，共2000字
```

### 关键信息

```
格式：要点式列表
- 每个要点一句话
- 重要性排序
- 最多5条
```

### 详细内容

```
格式：根据任务类型调整
- 文档写作：Markdown格式
- 代码：带注释的代码块
- 分析报告：带标题的章节
```

### 建议/后续步骤

```
格式：建议式列表
- 可操作的下一步
- 最多3条
- 非必需
```

## 示例

### 示例：标准写作任务

```
## 📋 执行摘要
完成了800字科幻短文创作，主题为2077年的北京，包含完整世界观设定。

---

## 📌 关键信息
- 🌃 场景：2077年北京，霓虹灯与胡同并存
- 🤖 主题：人工智能与人类共存
- 🔧 风格：硬科幻，强调技术细节
- 📖 结构：起承转合完整

---

## 📄 详细内容
[完整小说内容...]

---

## 💡 建议/后续步骤
- 可扩展为系列故事
- 可加入角色对话增强可读性
- 可调整风格为软科幻
```

## 格式规范

| 部分 | 格式 | 必填 |
|------|------|------|
| 执行摘要 | 简洁段落 | ✅ |
| 关键信息 | 列表 | ✅ |
| 详细内容 | 任务相关 | ✅ |
| 后续步骤 | 列表 | ❌ |
