# 分析模板 - 竞品分析报告

**模板 ID：** analysis-competitor  
**类型：** 分析  
**适用场景：** 竞品分析、市场调研、战略制定

---

## 任务描述

生成一份竞品分析报告

---

## Agent 角色配置

| 顺序 | 角色 | 职责 |
|------|------|------|
| 1 | market-researcher | 收集竞品信息 |
| 2 | data-analyst | 对比分析 |
| 3 | project-manager | 整理报告结构 |
| 4 | technical-writer | 撰写报告 |

---

## 输出要求

- 分析维度：产品、价格、渠道、推广
- 包含：SWOT 分析、竞争策略建议
- 格式：Markdown 报告

---

## Prompt 模板

```
请生成一份竞品分析报告：

**目标产品：** [用户指定]
**竞品列表：** [竞品 A、竞品 B、竞品 C]

**分析维度：**
1. 产品功能对比
2. 价格策略对比
3. 用户群体对比
4. 营销策略对比

**输出：**
- 竞品对比表格
- SWOT 分析
- 竞争策略建议
```
