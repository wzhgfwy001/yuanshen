# 翻译模板 - 技术文档翻译

**模板 ID：** translation-tech-doc  
**类型：** 翻译  
**适用场景：** 技术文档翻译、API 文档本地化

---

## 任务描述

翻译技术文档并保持专业性

---

## Agent 角色配置

| 顺序 | 角色 | 职责 |
|------|------|------|
| 1 | translator | 初稿翻译 |
| 2 | technical-writer | 技术术语校对 |
| 3 | translator | 润色优化 |
| 4 | test-engineer | 质量审核 |

---

## 输出要求

- 保持原文技术准确性
- 术语一致性
- 语言流畅自然
- 格式：Markdown

---

## Prompt 模板

```
请翻译以下技术文档：

**源语言：** 英文
**目标语言：** 中文

**文档内容：** [粘贴文档]

**要求：**
1. 保持技术术语准确性
2. 术语前后一致
3. 语言流畅自然
4. 保留代码和示例
5. 标注不确定的翻译

**输出格式：**
- 翻译稿
- 术语表
- 翻译注释
```
