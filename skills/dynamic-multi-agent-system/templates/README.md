## 🎯 使用方法

### 方式 1：直接使用模板

```
1. 选择适合的模板
2. 复制 Prompt 模板
3. 填写用户指定内容
4. 执行任务
```

### 方式 2：自动匹配（未来功能）

```
1. 用户描述任务
2. 系统自动匹配模板
3. 加载 Agent 配置
4. 执行任务
```
