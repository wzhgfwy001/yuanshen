## 🎯 使用方法

### 方式 1：使用模板管理器

```powershell
# 加载模板管理器
. "$PSScriptRoot\task-template-manager.ps1"

# 列出所有模板
Show-TemplateList

# 按分类查看
Show-TemplateList -category "writing"

# 查看模板详情
Show-TemplateDetail -id "article-daily-news"

# 搜索模板
Search-Templates -query "商业"

# 获取带参数的 Prompt
$result = Get-TemplatePrompt -id "article-daily-news" -params @{
    topic = "AI大模型最新进展"
    wordCount = 1000
    style = "专业严谨"
}
Write-Host $result.prompt
Write-Host "Agents: $($result.agents -join ', ')"
```

### 方式 2：直接使用模板文件

```powershell
# 查看模板文件
Get-Content "templates\writing-sci-fi-story.md"
```

## 📝 贡献新模板

**模板结构：**
```powershell
"template-id" = @{
    id = "template-id"
    category = "writing"              # 分类（见上方分类）
    name = "模板名称"
    description = "适用场景描述"
    version = "1.0.0"
    agents = @("role1", "role2")       # Agent 角色序列
    complexity = 3                    # 1-5 星
    estimatedTime = "15-20min"        # 预估时间
    prompt = @"
请创作一篇...                    # Prompt 模板
主题：{topic}                    # 占位符用 {key}
...
"@
    outputFormat = "markdown"          # 输出格式
    tags = @("标签1", "标签2")         # 搜索标签
}
```
