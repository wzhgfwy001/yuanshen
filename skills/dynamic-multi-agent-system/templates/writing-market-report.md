# 写作模板 - 市场分析报告

**模板 ID：** writing-market-report  
**类型：** 写作  
**适用场景：** 市场分析、行业报告、竞品分析

---

## 任务描述

生成一份专业的市场分析报告

---

## Agent 角色配置

| 顺序 | 角色 | 职责 |
|------|------|------|
| 1 | market-researcher | 收集市场数据 |
| 2 | data-analyst | 分析数据趋势 |
| 3 | technical-writer | 撰写报告 |
| 4 | test-engineer | 审核报告质量 |

---

## 输出要求

- 字数：3000-5000 字
- 格式：Markdown
- 包含：市场概况、竞品分析、趋势预测、建议

---

## Prompt 模板

```
请生成一份市场分析报告，要求：

1. 市场概况
   - 市场规模
   - 增长率
   - 主要参与者

2. 竞品分析
   - 竞品 A 分析
   - 竞品 B 分析
   - 竞争优势对比

3. 趋势预测
   - 短期趋势（1 年）
   - 长期趋势（3-5 年）

4. 建议
   - 市场进入策略
   - 风险规避建议
```
