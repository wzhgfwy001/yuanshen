### 详细描述（500-1000 字）

```
## 核心功能

混合动态多 Agent 协作系统是一个运行在 OpenClaw 之上的元级别协作框架，
主要解决复杂任务的自动化处理问题。系统能够智能识别任务类型，动态
组建专业 Agent 团队，通过并行/串行协作高效完成复杂任务。

### 8 大核心模块

1. **任务分类器** - 自动识别 4 种任务类型（单一/标准/创新/混合），
   识别准确率≥90%，平均置信度 0.92

2. **任务分解器** - 将复杂任务动态分解为 6-8 个可执行子任务，
   遵循正交性原则，依赖关系清晰

3. **执行协调器** - 协调多 Agent 并行/串行执行，支持最多 3 个主
   任务并发，12 个子 Agent 同时运行

4. **质量检查器** - 三层质量保障机制（自我检查→主 Agent 确认→
   独立审查），输出质量稳定达标（≥85 分）

5. **共享记忆层** - 多 Agent 协作的持久化存储，支持读写、版本
   控制、权限管理、原子操作

6. **多任务队列** - 智能任务调度，支持优先级（high/normal/low）、
   并发控制、超时重试

7. **Skill 固化追踪器** - 持续进化学习，3 次成功经验自动固化为
   可复用 Skill，版本管理（v1.0→v1.1→v2.0）

8. **用户反馈自动化** - 自动收集用户评分（1-5 分），低分触发改进，
   高分计入固化，生成满意度报告

### 技术优势

- **动态适配** - 不固定 Agent 数量，完全根据任务复杂度按需分配
- **质量优先** - 三层审查机制，每环节都有质量把关
- **持续进化** - 执行经验自动沉淀，系统能力持续提升
- **状态持久化** - 共享记忆层支持多 Agent 协作，避免信息孤岛
- **智能调度** - 并发控制、优先级调度、超时重试，系统稳定运行

### 应用场景

- 复杂创意写作（小说、剧本、系列文章）
- 多步骤数据分析与报告
- 完整项目开发（需求→设计→编码→测试）
- 研究与信息整理（多来源信息整合）
- 方案设计与规划（管理体系、营销策略等）
```
