## 📜 第一部分：世界观核心

### 1.1 时代背景（太阳系时代）
```
3026年，太阳系开发成熟

【势力分布】
• 东方联合国军
• 西方联合国军
• 各国星际舰队
• 星际海盗

【人类分布】
• 有人类一直生活在地球
• 有人类出生在外太空

【文化差异】
• 各势力理念不同，需要维稳
```

### 1.2 宇宙起源
- **大爆炸后**：外围先产生生命，能量耗尽后向中心逃离
- **地球角色**：外围生命逃离的能量补充站（灵魂能量收集器）
- **飞船形态**：九层宝塔，位于昆仑山脉
- **能量来源**：生命的灵魂能量（越开智越强）

### 1.3 灵魂能量（核心设定）
> **灵魂能量是宇宙中最顶级的能量形式**

| 能力 | 说明 |
|------|------|
| 影响空间 | 空间瞬移、空间折叠、空间门 |
| 影响时间 | 时间回溯（理论上） |
| 重塑物质 | 炼金、变形、元素操控 |
| 改变形态 | 物质转换 |
| 改变因果 | 规则系能力 |

**核心观点**：灵魂能量 = 影响宇宙的钥匙能量 = 科学（只是科技还无法解释）

### 1.4 宝塔与地球生命进化
```
抵达时 → 生命灵魂能量太薄弱 → 无法利用
    ↓
培育期 → 植物（基础）→ 动物（中等）→ 人类（最精纯）
    ↓
当代 → 人类站在地球之巅 → 支撑遗民永生 + 飞船动力
```

### 1.5 动植物成精
- **原因**：机缘巧合/开悟导致灵魂能量暴涨
- **结果**：自身发生质变成为"精"

### 1.6 法术 vs 魔法 统一解释
| 体系 | 形式 | 本质 |
|------|------|------|
| 中国法术 | 内丹、符箓、阵法等 | 灵魂能量的东方运用 |
| 西方魔法 | 元素、召唤、诅咒等 | 灵魂能量的西方运用 |
| 异能觉醒 | 觉醒本能释放 | 灵魂能量自发运用 |

## 👻 第三部分：遗民与三魂体系

### 3.1 遗民设定
| 项目 | 内容 |
|------|------|
| 来源 | 大爆炸后外围幸存的生命体 |
| 存在形式 | 灵魂（纯天魂） |
| 永生方式 | 以纯天魂形态存于天宫 |
| 特点 | 只有天魂，可申请投胎转世 |

### 3.2 三魂体系
| 魂 | 来源 | 作用 |
|----|------|------|
| 天魂 | 遗民灵魂下界投胎 | 带来潜能与天赋 |
| 地魂 | 人死后审判洗礼 | 因果清算、净化 |
| 人魂 | 与家族气运融合 | 家族传承、命运联结 |

### 3.3 三大闭环
**【闭环一：投胎】** 轮回殿 + 天魂 + 地魂 + 家族气运 → 新生命

**【闭环二：审判】**
```
大恶 → 化为纯能量供能
大善 → 保存灵魂印记，投胎聪慧
普通 → 正常轮回
```

**【闭环三：天兵天将】** 遗民天魂 + 后天成神 → 天庭编制补充

### 3.4 遗民升阶规则
```
普通遗民 → 申请投胎 → 做出贡献（如孔子开化万民）
    → 享受人间烟火 → 回归天庭升阶为神
```

### 3.5 封神榜体系
| 成神方式 | 说明 | 代表 |
|----------|------|------|
| 封神榜封禅 | 灵魂修炼体系被封神 | 姜子牙、李靖 |
| 肉身成圣 | 修炼极致直接成神 | 杨戬、哪吒 |
| 遗民投胎升阶 | 做出贡献升阶 | 孔子、老子 |

### 3.6 灵魂能量"黑市"（新增设定）
```
有人出卖"未来潜力"换取立刻的力量
    ↓
这就是为什么有些能力者越强越短命
    ↓
凌家在暗中打击这种交易
这是和暗流组织的另一层矛盾
```
