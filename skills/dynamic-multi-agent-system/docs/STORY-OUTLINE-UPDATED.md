## 🌍 世界观核心设定

### 宇宙起源理论
- **大爆炸后**：外围先产生生命，能量耗尽后向中心逃离
- **地球角色**：外围生命逃离的能量补充站（灵魂能量收集器）
- **飞船形态**：九层宝塔，位于昆仑山脉
- **能量来源**：生命的灵魂能量（越开智越强）

### 宝塔与地球生命的进化历程

```
【第一阶段：抵达】
宝塔抵达地球 → 发现存在原始生命
    ↓
问题：灵魂能量太薄弱，无法直接收集利用
    ↓
【第二阶段：培育期】
宝塔开始引导生命进化
    ↓
┌─────────────────────────────────────────────┐
│  植物（树木荣枯）→ 产生基础灵魂能量        │
│       ↓                                     │
│  动物（生死轮回）→ 能量更精纯              │
│       ↓                                     │
│  人类（智慧生命）→ 灵魂能量最精纯          │
│       ↓                                     │
│  【越开智的生物 = 越精纯的灵魂能量】       │
└─────────────────────────────────────────────┘
    ↓
【第三阶段：当代】
人类站在地球之巅
    ↓
灵魂能量充足 → 支撑遗民永生 + 飞船动力
```

### 灵魂能量（核心设定）

**灵魂能量是宇宙中最顶级的能量形式：**

```
【灵魂能量的能力】
• 影响空间 → 空间瞬移、空间折叠、空间门
• 影响时间 → 时间回溯（理论上）
• 重塑物质 → 炼金、变形、元素操控
• 改变形态 → 物质转换
• 改变因果 → 规则系能力
    ↓
【核心观点】
灵魂能量 = 影响宇宙的钥匙能量
```

**这就是超能力的来源：**
```
人类觉醒灵魂能量
    ↓
空间系能力：空间瞬移、空间感知、空间门
时间系能力：时间操控（理论）
物质系能力：炼金、变形、元素操控
规则系能力：改变因果律
```

**植物/动物成精：**
```
机缘巧合 / 意外开悟
    ↓
灵魂能量暴涨
    ↓
自身发生质变 → "精"
    ↓
例：人参精、狐妖等
```

**法术 vs 魔法（统一解释）：**
| 体系 | 形式 | 本质 |
|------|------|------|
| 中国法术 | 内丹、外丹、符箓、阵法等 | 灵魂能量的东方运用方式 |
| 西方魔法 | 元素魔法、召唤、诅咒等 | 灵魂能量的西方运用方式 |
| 异能 | 觉醒异能 | 灵魂能量本能释放 |

**核心观点：**
```
灵魂能量 = 科学
只是当前的科学技术还无法解释而已
```

**核心逻辑：**
| 生命阶段 | 灵魂能量 | 说明 |
|----------|----------|------|
| 原始生命 | 极薄弱 | 无法利用 |
| 植物 | 基础 | 树木荣枯产生 |
| 动物 | 中等 | 生死轮回 |
| 人类 | 最精纯 | 智慧生命，地球之巅 |

**这就是为什么：**
- 地球古代有修仙传说（宝塔培养生命）
- 中国神话有封神体系（灵魂能量进化）
- 人类成为最强大的灵魂能量来源

### 近1000年的重大变化（关键设定）
> **为什么现在才揭露神秘面纱？**

| 原因 | 说明 |
|------|------|
| **宇宙外层文明入侵** | 探测到地球位置，准备入侵 |
| **灵魂能量危机** | 遗民永生消耗 + 能量补充不足，宝塔能量日益紧张 |
| **飞船任务转变** | 从"循序引导地球文明"转为"快速培养精英迎战" |
| **加速揭露** | 近1000年开始对少数人揭露宝塔存在 |
| **目的** | 在入侵者到来前，培养足够多的强者 |

**隐藏危机：**
- 遗民以灵魂形态永生，但能量消耗持续
- 地球灵魂能量补充不足（人口基数问题）
- 需要快速培养强大的灵魂收割者/能力者来应对危机

| 体系 | 神系来源 | 轮回认知 | 实际情况 |

**西方神话真相：**
```
西方宗教 → 想象神祇（宙斯、奥丁、上帝等）
    ↓
【误解】以为是真正的神
    ↓
【可能的真相】
• 极少数：天魂投胎在西方展示能力 → 被当神崇拜
• 大多数：纯粹的宗教想象
    ↓
【但无论如何】
死后灵魂 → 最终都进入宝塔轮回体系
    ↓
只是西方民众不知道这个真相
```

**核心观点：**
- 西方神话中的神 ≠ 宝塔真神
- 西方神话是宗教自我演化的结果
- 他们的"神"可能是天魂展示能力被崇拜
- 但最终归宿都一样：宝塔轮回

## ⚠️ 势力关系

### 昆仑宝塔 → 华夏联盟
- 凌家（核心家族，真正知道宝塔）
- 宁家（核心家族，真正知道宝塔）
- 龙腾学院（培养基地）
- 官方特勤组

### 昆仑宝塔 → 西方教廷
- 与凌家矛盾激化
- 教皇厅、圣堂骑士团、魔法议会
- **矛盾深层原因：**
  - 表面：势力争夺、教皇儿子被杀
  - 深层：教廷觉察到宝塔存在但无法面对
- **宁月初记恨**：生产时被骚扰，记下仇恨，在凌午夜入学后报复

### 昆仑宝塔 → 暗流组织

**起源（重要设定）：**
```
宇宙外围生命探测到地球
    ↓
宝塔发现并预警
    ↓
这些生命派前哨入侵地球
    ↓
宝塔反击，前哨大部分被消灭
    ↓
残余势力潜伏下来 → 演变成【暗流组织】
    ↓
【破壁者】【灵魂收割者】【叛逃者】= 前哨余孽
```

**暗流组织本质：**
| 身份 | 说明 |
|------|------|
| **来源** | 入侵者前哨的残余势力 |
| **任务** | 为真正入侵做准备 |
| **手段** | 潜伏、渗透、灵魂收割 |
| **难度** | 盘根错节、隐藏在暗处，难以发掘 |

**与西方的联系：**
```
暗流组织（入侵者余孽）
    ↓
潜伏西方多年
    ↓
与西方教廷产生千丝万缕联系
    ↓
可能是：
• 教廷内部被渗透
• 教廷利用暗流对付华夏
• 教廷不知情被利用
```

**这就是为什么：**
- 暗流组织如此强大又如此隐蔽
- 教廷与凌家对抗的深层原因
- 宇宙入侵线提前埋下的伏笔
