- `SubAgentInfo` - 子 Agent 信息数据类
- `TaskStatus` - 任务状态枚举

**系统限制：**
```python

MAX_SUBAGENTS = 12            # 最大子 Agent 总数
MAX_SUBAGENTS_PER_TASK = 6    # 单任务最大子 Agent 数
```
