## 🔄 伏笔回收规则

```
1. 伏笔必须在指定章节前埋入
2. 回收时必须有逻辑铺垫
3. 不能突然出现毫无铺垫的能力
4. 跨阶段伏笔必须有递进关系
```

---

## ❌ 常见错误

```
❌ 伏笔埋了没回收
❌ 回收太突然，没有铺垫
❌ 伏笔之间相互矛盾
❌ 埋入的伏笔超出角色认知范围
```
