## 🎯 学习重点

### 1. 设定一致性原则

```
核心规则：
├── 一致性 > 丰富性
├── 新设定不能破坏旧设定
├── 设定要有边界限制
└── 解释要符合已有逻辑

检查流程：
新设定 → 对比旧设定 → 无冲突 → 可用
新设定 → 发现冲突 → 修改/放弃
```

### 2. 设定层次结构

```
第一层：核心设定（不可改）
├── 灵魂能量本质
├── 九层宝塔结构
└── 三魂闭环

第二层：重要设定（需谨慎修改）
├── 能力等级
├── 势力关系
└── 吞噬者特性

第三层：边缘设定（可扩展）
├── 具体城市
├── 具体人物
└── 日常细节
```

### 3. 设定呈现技巧

```
❌ 生硬解释：
"灵魂能量是宇宙中最顶级的能量形式，可以影响空间、时间..."

✅ 自然呈现：
"凌午夜感觉到一股力量在体内流动，那是..."
（通过角色体验展示设定）
```

## 📚 案例分析

### 成功案例：哈利波特魔法规则

```
优点：
- 魔法有代价（咒语、手势）
- 规则有限制（麻瓜无法使用）
- 细节丰富（魔杖材质、杖芯）

我们的借鉴：
- 能力有代价和限制
- 不是所有人都能觉醒
- 有具体的体系细节
```

### 失败案例：战力崩坏

```
常见问题：
❌ 设定S级，后面出现SS级无解释
❌ 前文说能量不够，后面无限使用
❌ 能力没有边界

解决方法：
- 设定写在NOVER-REFERENCE.md
- 严格遵守禁止事项
- 新能力必须有铺垫
```
