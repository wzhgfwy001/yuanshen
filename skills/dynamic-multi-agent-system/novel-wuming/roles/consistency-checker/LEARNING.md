## 🎯 学习重点

### 1. 五维检查法

```
时间线检查：
├── 时间是否连续
├── 季节/天气是否矛盾
├── 人物位置是否合理
└── 事件顺序是否正确

能力检查：
├── 能力是否超出当前等级
├── 能力使用有无代价
├── 能力限制是否遵守
└── 能力展示有无铺垫

关系检查：
├── 称呼是否正确
├── 关系变化有无铺垫
├── 情感发展是否合理
└── 势力关系是否矛盾

设定检查：
├── 新设定是否与旧设定冲突
├── 细节是否前后一致
├── 世界观是否被破坏
└── 禁止事项是否违反

伏笔检查：
├── 伏笔是否埋入
├── 回收是否有铺垫
├── 伏笔之间是否矛盾
└── 新伏笔是否合理
```

### 2. 常见漏洞清单

```
【时间线漏洞】
❌ 昨天还在打架，今天就称兄道弟
❌ 冬天突然穿短袖没解释
❌ 上一秒在这里，下一秒就到别处

【能力漏洞】
❌ S级能力随意使用
❌ 能力突然变强没解释
❌ 前文说能量不够，后面无限用

【关系漏洞】
❌ 上章还叫"敌人"，这章就叫"哥"
❌ 从没见过面，突然很熟

【世界观漏洞】
❌ 宝塔层级改变
❌ 灵魂能量规则改变
```

### 3. 高效检查技巧

```
第一步：快速扫读（5分钟）
└── 找出明显问题

第二步：逐章精读（15分钟）
└── 逐句检查逻辑

第三步：对照锚定文档
└── 检查设定一致性

第四步：检查伏笔
└── 伏笔追踪表对照

第五步：复查疑点
└── 标记有问题的地方
```

## 📝 每次创作检查清单

```
【时间线】
- [ ] 季节/天气正确
- [ ] 人物位置合理
- [ ] 事件顺序正确

【能力】
- [ ] 无超纲能力展示
- [ ] 能力有代价/限制
- [ ] 能力展示有铺垫

【关系】
- [ ] 称呼正确
- [ ] 关系变化有铺垫

【伏笔】
- [ ] 已埋伏笔正确回收
- [ ] 新伏笔合理
- [ ] 无矛盾伏笔

【禁止事项】
- [ ] 无违反禁止事项
```

## 📚 漏洞案例库

### 时间线漏洞

```
❌ 案例1：
"昨天，他们还是敌人。今天，他们成了朋友。"
问题：关系变化太快，没铺垫

❌ 案例2：
"冬天的早晨，窗外的蝉鸣不断。"
问题：季节错误

❌ 案例3：
"凌午夜刚还在操场，下一秒就出现在教室。"
问题：没有交代瞬移/能力
```

### 能力漏洞

```
❌ 案例1：
"凌午夜使出了SS级能力..."
问题：第一部禁止SS级

❌ 案例2：
"他使用了时间回溯..."
问题：时间能力暂未揭示

❌ 案例3：
"吞噬者被一击毙命。"
问题：只能驱逐，不能击杀
```
