## 🎯 学习重点

### 1. 三幕结构（必须掌握）

```
第一幕（30%）：建置
├── 开场钩子（10%）
├── 主题揭示
├── 设定展示
└── 第一情节点（30%）

第二幕（50%）：对抗
├── 上升动作
├── 中点转折
├── 下降动作
└── 第二情节点（75-80%）

第三幕（20%）：解决
├──  climax
└── 结局
```

### 2. 章节衔接技巧

```
好衔接：
├── 场景转换（时间/地点）
├── 情绪延续
├── 伏笔回收
└── 悬念钩子

坏衔接：
├── 突然跳转
├── 情绪断层
└── 逻辑断裂
```

### 3. 节奏把控

```
快节奏：动作、对话、紧张场面
慢节奏：描写、内心、过渡

正确比例：
├── 战斗/冲突：快
├── 日常/过渡：慢
├── 高潮：先慢后快
└── 结尾：留白
```

## 📝 每次创作检查清单

```
【章节衔接】
- [ ] 开头承接上章结尾
- [ ] 场景转换自然
- [ ] 情绪连贯

【节奏】
- [ ] 没有连续3段以上慢节奏
- [ ] 高潮前有铺垫
- [ ] 结尾有钩子

【悬念】
- [ ] 每章至少1个悬念
- [ ] 悬念有回收
- [ ] 新旧悬念衔接
```

---

## 📚 案例分析

### 成功案例：《凡人修仙传》开篇

```
结构分析：
第1章：危机开场（被追杀）
第2章：转机出现（意外得宝）
第3章：新挑战（宗门选择）

衔接技巧：
- 情绪连续
- 悬念递进
- 每章结尾有钩子
```

### 失败案例：常见新手错误

```
❌ 第一章写太多设定
❌ 章节之间无情绪衔接
❌ 高潮没有铺垫直接来
❌ 结尾没有钩子
```
