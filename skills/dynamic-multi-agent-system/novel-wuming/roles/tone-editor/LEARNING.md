## 🎯 学习重点

### 1. AI味完整清单

```
【描写类】
❌ "眼中闪过...的光芒"
❌ "心中涌起...的情绪"
❌ "嘴角微微上扬/抽搐"
❌ "脸色苍白/铁青"
❌ "周身散发着...的气息"
❌ "完美的笑容"
❌ "英俊的脸庞"

【动作类】
❌ "他优雅地..."
❌ "动作行云流水"
❌ "仿佛...一般"

【心理类】
❌ "他心想..."
❌ "内心深处..."
❌ "涌起一股...的感情"

【对话类】
❌ 完整的客套话
❌ 过于有逻辑的表达
❌ 废话连篇
❌ 总是回应对方

【反应类】
❌ 总是恰当的反应
❌ 永远冷静/永远激动
❌ 从不犯错
```

### 2. 改稿技巧

```
改稿顺序：
1. 第一遍：去AI味（最优先）
2. 第二遍：统一文风
3. 第三遍：检查情感基调
4. 第四遍：细节打磨

改稿原则：
├── 大段删除 > 小段修改
├── 用动作替代心理
├── 用沉默替代千言万语
├── 用不完美替代完美
└── 用口语替代书面语
```

### 3. 文风统一规范

```
《吾名午夜》文风：
├── 简洁有力
├── 短句为主
├── 动作描写优先
├── 对话简短
├── 心理描写少
├── 留白要多
└── 不要华丽辞藻

参考风格：
├── 武侠小说的简洁
├── 严肃文学的质感
└── 网络文学的节奏
```

## 📝 改稿检查清单

```
【AI味】
- [ ] 无"光芒/情绪"描写
- [ ] 无"完美/优雅"形容
- [ ] 无过度心理描写

【对话】
- [ ] 简短有力
- [ ] 打断/沉默
- [ ] 口语化

【动作】
- [ ] 动作具体
- [ ] 小动作多
- [ ] 无"行云流水"

【情感】
- [ ] 克制
- [ ] 留白
- [ ] 少用感叹号
```

## 📚 改稿案例库

### 案例1：心理改动作

```
❌ 改前：
凌午夜感到一阵恐惧，心跳加速，手心出汗。

✅ 改后：
凌午夜的呼吸停了一拍。
手心湿了。
他想跑。
```

### 案例2：对话改沉默

```
❌ 改前：
"我真的不是故意的，你相信我吗？"东方凌薇急切地解释。
"我理解你的感受，但..."

✅ 改后：
"我..."
东方凌薇话没说完。
凌午夜已经转身走了。
```

### 案例3：完美改缺陷

```
❌ 改前：
凌午夜冷静地分析了情况，迅速做出了正确的判断。

✅ 改后：
凌午夜脑子一片空白。
他不知道该怎么办。
腿有点软。
```
