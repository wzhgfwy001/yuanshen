## ⚠️ 当前薄弱点

```
1. 称呼容易混乱（凌午夜叫东方凌薇"东方"）
2. 性格不够立体（过于完美）
3. 对话缺乏个性（口癖不明显）
4. 关系变化没有铺垫
```

## 🎯 学习重点

### 1. 称呼规范（必须烂熟于心）

```
学院小队内部：
├── 凌午夜 → "午夜"（队友叫）
├── 东方凌薇 → "凌薇"或"薇薇"（男友叫）
├── 南宫墨 → "墨哥"
├── 西门吹雪 → "吹雪"
├── 北冥瑶 → "瑶儿"
└── 中原星辰 → "星辰"

华夏联盟（凌家/宁家）：
├── 家主 → "家主"
├── 同辈 → "哥/姐"
└── 长辈 → "叔/伯"

教廷：
├── 教皇 → "教皇"
├── 主教 → "大人"
└── 骑士长 → "骑士长"
```

### 2. 性格塑造技巧

```
❌ AI味性格（太完美）：
- 勇敢、聪明、善良、帅气
- 反应总是正确
- 说话总是有道理

✅ 真实性格（有缺陷）：
- 凌午夜：沉默、不善表达、但关键时刻靠得住
- 南宫墨：冷静、毒舌、但其实很在意队友
- 东方凌薇：理性、但对凌午夜会感性
```

### 3. 对话口癖设计

```
凌午夜：
- "嗯。"
- "知道了。"
- "走。"
- （很少主动说话）

南宫墨：
- "无聊。"
- "这种程度..."
- "真是浪费时间。"

东方凌薇：
- "午夜你看！"
- "我觉得..."
- （话比较多）

西门吹雪：
- 几乎不说话
- 说话也很简短
```

## 📝 每次创作检查清单

```
【称呼】
- [ ] 所有称呼符合规范表
- [ ] 没有写错角色名字
- [ ] 势力内部称呼正确

【性格】
- [ ] 性格与设定一致
- [ ] 有缺陷/弱点
- [ ] 反应符合性格

【对话】
- [ ] 有口癖/说话风格
- [ ] 对话简短有力
- [ ] 没有废话
```

## 📚 案例分析

### 成功案例：《斗破苍穹》萧炎

```
性格塑造：
- 初期：天才陨落、隐忍
- 中期：复仇驱动、有谋略
- 后期：王者风范

对话特点：
- 痞气但不恶心
- 关键时刻正经
- 对敌人冷酷
```

### 失败案例：完美主角

```
❌ 所有对话都是道理
❌ 对谁都彬彬有礼
❌ 没有情绪波动
❌ 永远不会错
```
