## 🎯 学习重点

### 1. 去AI味核心技巧

```
AI味特征：
├── "眼中闪过...的光芒"
├── "心中涌起...的情绪"
├── "嘴角微微上扬"
├── 对话过于完整有逻辑
└── 反应总是恰到好处

去除方法：
├── 动作代替心理描写
├── 沉默代替千言万语
├── 不完美才是真实
└── 口语化对话
```

### 2. 对话写作技巧

```
❌ AI味对话：
"我理解你的感受，我们一起努力吧！"
"这个问题确实很困难，但我们应该冷静分析..."

✅ 真实对话：
"......"（沉默）
"闭嘴。"
"嗯。"

技巧：
- 打断别人
- 说一半不说了
- 用动作回应
- 说废话/跑题
```

### 3. 场景描写技巧

```
❌ AI味描写：
"教室里弥漫着紧张的气氛，同学们都屏住呼吸等待结果。"

✅ 真实描写：
"教室很安静。有人在喝水，有人在转笔，有人在发呆。"

技巧：
- 具体细节 > 概括描写
- 感官 > 情绪
- 动态 > 静态
```

## 📝 去AI味检查清单

```
【对话】
- [ ] 没有完整的客套话
- [ ] 打断/沉默/说一半
- [ ] 口语化

【描写】
- [ ] 没有"完美"形容词
- [ ] 动作代替心理
- [ ] 具体细节

【人物】
- [ ] 反应有个性
- [ ] 不总是对的
- [ ] 有小动作
```

## 📚 案例对比

### 去AI味前后对比

```
【场景：凌午夜被问到感受】

❌ AI味：
凌午夜的眼中闪过一丝复杂的光芒，那是愤怒与悲伤交织的情绪。
"我一定会找到真相的。"他握紧拳头，声音低沉而坚定。

✅ 去AI味：
凌午夜没说话。
手攥着，指节发白。
"......"

【场景：东方凌薇发现异常】

❌ AI味：
"午夜，你看那边！"东方凌薇惊讶地指着远处，眼睛瞪得圆圆的。

✅ 去AI味：
"午夜。"
东方凌薇拽了拽他的袖子，下巴朝右边点了点。
没说话，但眼神不对。
```
