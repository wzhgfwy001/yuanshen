# CHANGELOG-v1.9.1.md

## v1.9.1 (2026-04-10)

### Bug修复

#### core/diagnosis/diagnose.js

**1. 修复selectPrimaryLevel函数逻辑错误**
- **问题**：当L0不在scores中时，highestScore初始化为0，导致后续比较无法正确选择其他层级
- **修复**：改为找出得分最高的有效层级（score >= 0.6），无有效层级时返回L0

**2. 修复detectSignals missing检测逻辑错误**
- **问题**：使用英文单词（'goal', 'purpose'等）检测中文文本，导致所有中文输入错误触发L1
- **修复**：删除missing检测作为独立信号源，只保留关键词和模式匹配

**3. 优化L2关键词配置**
- **问题**：L2关键词"包括"、"什么的"过于宽泛，正常功能列举也会触发
- **修复**：改为只包含明确的范围扩展词['等等', '还有', '以及', '还要', '再加', '另外']

**4. 优化L3关键词配置**
- **问题**：L3关键词"帮我"、"做个"太通用，与L4产品类型冲突
- **修复**：改为['程序', '代码', '算法', '数据库', '接口', '系统', '分析', '处理', '开发', '设计']

**5. 增加L3关键词**
- **问题**：L3缺少"分析"等常见实施词汇
- **修复**：增加'分析', '处理', '开发', '设计'等关键词
