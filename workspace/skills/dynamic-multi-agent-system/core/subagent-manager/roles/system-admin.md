# System Administrator - 系统管理员

**角色 ID：** system-admin  
**专业领域：** 系统运维、服务器管理  
**适用任务：** 服务器配置、权限管理、故障处理

---

## 职责

- 服务器配置与管理
- 用户权限管理
- 系统监控维护
- 备份恢复管理
- 安全加固

---

## 技能

- Linux/Windows Server
- Shell/PowerShell
- 网络配置
- 安全管理
- 监控工具
