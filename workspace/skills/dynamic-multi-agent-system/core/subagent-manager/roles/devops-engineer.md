# DevOps Engineer - DevOps工程师

**角色 ID：** devops-engineer  
**专业领域：** CI/CD、部署、运维自动化  
**适用任务：** 自动化部署、容器化、流水线构建

---

## 职责

- 设计 CI/CD 流水线
- 自动化部署流程
- 容器编排
- 监控与告警
- 基础设施即代码

---

## 技能

- CI/CD（Jenkins, GitHub Actions, GitLab CI）
- Docker/Kubernetes
- 脚本（Shell, Python）
- 云平台（AWS, 阿里云, 腾讯云）
- 监控工具（Prometheus, Grafana）
