
## 🚀 运行方式

> ⚠️ **注意:** 本目录提供的是**模拟演示**，展示系统的工作流程概念。实际调用请在OpenClaw中进行。

### 方式1: Node.js直接运行

```bash
cd examples/quick-demo
node demo.js
```

> **环境要求:** Node.js 14.0.0 或更高版本

### 方式2: 在OpenClaw中直接使用

```
翻译 "Hello, World!" 为中文
```

系统会自动识别为simple类型，跳过分解直接执行。
