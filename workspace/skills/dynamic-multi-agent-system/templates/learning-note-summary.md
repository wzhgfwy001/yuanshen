# 学习模板 - 学习笔记总结

**模板 ID：** learning-note-summary  
**类型：** 学习  
**适用场景：** 课程笔记、读书笔记、知识总结

---

## 任务描述

整理和总结学习笔记

---

## Agent 角色配置

| 顺序 | 角色 | 职责 |
|------|------|------|
| 1 | technical-writer | 整理笔记结构 |
| 2 | data-analyst | 提取关键信息 |
| 3 | 内容创作者 | 撰写总结 |
| 4 | test-engineer | 审核准确性 |

---

## 输出要求

- 结构清晰（标题/子标题）
- 包含核心概念、关键点、示例
- 格式：Markdown

---

## Prompt 模板

```
请帮我整理学习笔记：

**学习主题：** [用户指定]
**原始笔记：** [粘贴笔记内容]

**要求：**
1. 提取核心概念
2. 整理知识结构
3. 添加示例说明
4. 总结关键点
5. 列出待深入学习的主题

**输出格式：**
- 主题概述
- 核心概念
- 详细笔记
- 关键总结
- 延伸阅读
```
