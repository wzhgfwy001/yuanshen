# 🚀 阳神系统快速启动

**适用场景：** 想快速启动任务，不需要了解完整系统

---

## 📌 一句话启动（最简单）

直接描述你的需求，系统自动判断任务类型并选择最优执行方案。

### 示例
```
帮我写一篇产品介绍，300字
帮我分析这段代码的问题
写一个10章的悬疑小说
帮我调研一下新能源汽车市场
```

---

## 🎯 场景化启动模板

### 1. 写作任务

**模板：**
```
写一篇[类型]文章：
- 主题：[主题]
- 字数：[字数]
- 风格：[风格]
```

**示例：**
```
写一篇产品评测文章：
- 主题：iPhone 15 Pro
- 字数：1000字
- 风格：客观中立
```

---

### 2. 代码任务

**模板：**
```
用[语言]开发[功能]：
- 需求：[具体需求]
- 框架：[如有指定]
```

**示例：**
```
用Python开发一个待办事项CLI工具：
- 需求：添加/删除/查看任务
- 框架：只用标准库
```

---

### 3. 分析任务

**模板：**
```
分析[对象]：
- 维度：[分析维度]
- 输出：[报告/表格/图表]
```

**示例：**
```
分析竞品微信小程序：
- 维度：功能、用户体验、定价
- 输出：对比表格 + 建议
```

---

### 4. 创意任务

**模板：**
```
创作[类型]：
- 主题：[主题]
- 风格：[风格]
- 字数：[字数]
```

**示例：**
```
创作一个2分钟的品牌故事：
- 主题：咖啡品牌创业故事
- 风格：温暖感人
- 字数：400字
```

## ⚡ 高级启动（一句话搞不定时用）

当任务复杂或有特殊要求时，使用结构化启动：

```
任务类型：[单一/标准/创新/复杂]
任务描述：[详细描述]
约束条件：[字数/时间/风格/预算等]
输出要求：[格式/内容要求]
```

**示例：**
```
任务类型：创新任务
任务描述：为一个00后用户群体设计社交App
约束条件：预算50万，3个月内上线
输出要求：包含功能列表、技术方案、市场分析
```

## 🔧 任务类型说明

| 类型 | 复杂度 | 适用场景 | Agent数量 |
|------|--------|---------|-----------|
| 单一任务 | ⭐ | 简单查询/计算/翻译 | 1个 |
| 标准任务 | ⭐⭐ | 需要2-3步处理 | 2-3个 |
| 创新任务 | ⭐⭐⭐ | 创作/分析/规划 | 3-5个 |
| 复杂任务 | ⭐⭐⭐⭐ | 多角色协作/长篇创作 | 5+个 |

---

## 💡 提示

1. **越具体越好** — "写一篇科技文章" vs "写一篇关于ChatGPT对教育影响的文章"
2. **说明用途** — "写给投资人看的商业计划" vs "随便写写"
3. **指定格式** — "输出JSON格式" vs "随便输出"
4. **设定限制** — "不用AI相关的词" vs "不要限制"
