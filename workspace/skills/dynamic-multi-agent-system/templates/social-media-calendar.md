# 社交媒体模板 - 内容日历规划

**模板 ID：** social-media-calendar  
**类型：** 社交媒体  
**适用场景：** 内容规划、社媒运营、品牌营销

---

## 任务描述

制定社交媒体内容日历

---

## Agent 角色配置

| 顺序 | 角色 | 职责 |
|------|------|------|
| 1 | social-media-specialist | 制定内容策略 |
| 2 | 内容创作者 | 创作内容创意 |
| 3 | project-manager | 规划时间线 |
| 4 | data-analyst | 分析最佳发布时间 |

---

## 输出要求

- 周期：周/月
- 平台：微信/微博/抖音（可选）
- 包含：主题、文案、发布时间

---

## Prompt 模板

```
请制定社交媒体内容日历：

**品牌/产品：** [用户指定]
**周期：** 一个月
**平台：** 微信、微博、抖音

**要求：**
1. 每周主题规划
2. 每日内容创意
3. 文案草稿
4. 最佳发布时间建议
5. KPI 指标设定

**输出格式：**
- 内容日历表格
- 文案示例
- 发布建议
```
