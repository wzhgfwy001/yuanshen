# 代码模板 - Bug 调试与修复

**模板 ID：** code-bug-fix  
**类型：** 代码  
**适用场景：** Bug 修复、代码调试、错误排查

---

## 任务描述

分析并修复代码中的 Bug

---

## Agent 角色配置

| 顺序 | 角色 | 职责 |
|------|------|------|
| 1 | test-engineer | 复现 Bug |
| 2 | 代码开发者 | 定位问题 |
| 3 | 代码开发者 | 修复代码 |
| 4 | test-engineer | 验证修复 |

---

## 输出要求

- 分析 Bug 原因
- 提供修复方案
- 输出修复后的代码
- 添加测试用例

---

## Prompt 模板

```
请帮我调试并修复以下代码：

**问题描述：** [用户描述]
**错误信息：** [如有]
**代码：** [粘贴代码]

**要求：**
1. 分析 Bug 根本原因
2. 提供修复方案
3. 输出修复后的完整代码
4. 添加测试用例验证修复
5. 给出预防建议
```
