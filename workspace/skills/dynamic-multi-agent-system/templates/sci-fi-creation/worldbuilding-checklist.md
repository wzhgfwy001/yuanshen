# 科幻世界观检查清单

## 科技水平

- [ ] 明确时代背景（年份、地点）
- [ ] 核心科技描述清晰
- [ ] 科技水平前后一致
- [ ] 无时代错乱元素
- [ ] 科技设定服务于故事

## 社会结构

- [ ] 社会制度描述清晰
- [ ] 阶级/权力结构合理
- [ ] 经济体系自洽
- [ ] 文化与科技水平匹配

## 核心冲突

- [ ] 主要矛盾清晰
- [ ] 冲突来源合理
- [ ] 与科技设定相关
- [ ] 能推动情节发展

## 细节丰富度

- [ ] 日常生活描写具体
- [ ] 技术细节可信
- [ ] 环境描写有科幻感
- [ ] 人物行为符合设定

## 逻辑自洽

- [ ] 设定无明显漏洞
- [ ] 因果关系合理
- [ ] 无自相矛盾
- [ ] 符合内部逻辑

## 评分

□ 优秀(5) □ 良好(4) □ 合格(3) □ 需改进(2) □ 不合格(1)
