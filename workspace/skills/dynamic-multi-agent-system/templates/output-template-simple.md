# 简单任务输出模板

## 适用场景

- 类型1：单一任务 (Simple Task)
- 诊断结果：L0 - 执行明确
- 复杂度：低

## 输出原则

**直接输出，无需额外结构**

```
✓ 直接给出结果
✓ 简洁明了
✓ 无需解释过程（除非用户要求）
```

## 示例

### 示例1：翻译任务

```
输入：翻译 "Hello, world!" 为中文
输出：你好，世界！
```

### 示例2：计算任务

```
输入：计算 2 + 2
输出：4
```

### 示例3：总结任务

```
输入：总结这篇文章的主要观点
输出：[直接输出总结内容]
```

## 格式规范

| 情况 | 输出格式 |
|------|---------|
| 纯文本 | 直接文本 |
| 数字/计算 | 直接数字或结果 |
| 代码 | 代码块 |
| 列表 | 无序列表或有序列表 |
| 表格 | Markdown表格 |

## 禁止事项

- ❌ 不要说"好的，我来帮您..."
- ❌ 不要解释执行步骤
- ❌ 不要添加额外的说明
- ❌ 不要询问确认（除非有问题）
