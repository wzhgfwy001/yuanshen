# 悬疑小说检查清单

## 案件设计

- [ ] 作案动机合理
- [ ] 作案手法可行
- [ ] 密室诡计无漏洞
- [ ] 时间线清晰一致
- [ ] 线索布局合理

## 悬念设置

- [ ] 开篇有吸引力
- [ ] 中段有转折
- [ ] 信息控制得当
- [ ] 读者好奇心持续

## 伏笔埋设

- [ ] 关键伏笔已埋设
- [ ] 伏笔隐蔽但可追溯
- [ ] 无突兀的"天降神兵"
- [ ] 结局可回溯理解

## 人物塑造

- [ ] 凶手动机充分
- [ ] 侦探/主角能力合理
- [ ] 配角有存在价值
- [ ] 人物行为符合设定

## 逻辑一致性

- [ ] 时间线无矛盾
- [ ] 空间关系清晰
- [ ] 证据链完整
- [ ] 推理过程严谨

## 节奏控制

- [ ] 张弛有度
- [ ] 高潮设置合理
- [ ] 结局收束完整
- [ ] 无拖沓或仓促

## 评分

□ 优秀(5) □ 良好(4) □ 合格(3) □ 需改进(2) □ 不合格(1)
