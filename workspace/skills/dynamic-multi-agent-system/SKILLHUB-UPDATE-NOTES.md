## ❓ 常见问题（FAQ）

```
Q1: 优化功能会影响输出质量吗？
A: 不会。优化只是智能选择模型和处理策略，输出质量由模型保证，且通过渐进式处理和质量检查机制，质量反而更稳定。

Q2: Token 节省是怎么实现的？
A: 通过 8 大优化模块：缓存避免重复、模型按需选择、批量合并调用、渐进式避免返工、预算控制防止超支、智能重试失败恢复、结果复用历史数据。

Q3: 我需要手动配置优化参数吗？
A: 不需要。系统默认启用最优配置，自动根据任务类型选择策略。高级用户可在配置文件中自定义。

Q4: 缓存会保存多久？
A: 默认保存 30 天，或直到缓存达到上限（10000 条）。可自定义配置。

Q5: 如果预算设置过低怎么办？
A: 系统会智能降级到轻量模型保证任务完成，不会中断。同时会发出告警提示调整预算。

Q6: 结果复用会泄露隐私吗？
A: 不会。所有数据存储在本地，不会上传到外部服务器。可设置自动清理策略。

Q7: 优化功能会增加延迟吗？
A: 不会。缓存命中时直接返回（0 延迟），批量处理反而减少总延迟。整体速度提升 3-8 倍。

Q8: 适合什么类型的任务？
A: 所有类型都适合。简单任务自动用轻量模型，复杂任务用强力模型，相似任务自动复用，重复查询直接缓存。
```
