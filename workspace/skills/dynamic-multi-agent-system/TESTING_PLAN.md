## 问题分类

### 严重问题（P0）

- 系统崩溃
- 子 Agent 无法创建
- 输出完全不符合要求
- 资源泄漏

### 一般问题（P1）

- 任务分类错误
- 输出质量不稳定
- 执行时间过长
- 部分功能异常

### 轻微问题（P2）

- 文档不完善
- 提示信息不友好
- 性能可优化
- 用户体验问题

---

## 通过标准

**系统通过测试的条件：**

1. 所有测试任务完成
2. 无严重问题（P0）
3. 一般问题（P1）< 5 个
4. 输出质量符合要求
5. 资源正确清理
