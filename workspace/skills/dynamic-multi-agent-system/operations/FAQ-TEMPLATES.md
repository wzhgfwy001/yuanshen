## 🎯 使用操作类

### Q3: 如何调用多 Agent 协作？
**回复：**
```
使用方法超简单：
1. 直接描述你的复杂任务（如「帮我写一份市场分析报告」）
2. 系统会自动识别任务类型并组建 Agent 团队
3. 等待交付成果即可！

系统支持 4 种任务类型，自动适配最优策略。
```

### Q4: 如何查看任务执行状态？
**回复：**
```
查看任务状态：
1. 运行监控大屏：cd skills/dynamic-multi-agent-system/core/multi-task-queue && start-dashboard-simple.bat
2. 访问 http://localhost:5000/dashboard-simple.html
3. 实时查看任务队列和 Agent 状态

每 5 秒自动刷新，一目了然！
```

### Q5: 支持哪些任务类型？
**回复：**
```
系统支持 4 种任务类型：

🔹 单一任务 - 简单查询/计算（1 个 Agent）
🔹 标准任务 - 2-3 步处理（2-3 个 Agent）
🔹 创新任务 - 创作/分析/规划（4-6 个 Agent）
🔹 复杂任务 - 多步骤协作（6+ 个 Agent）

系统会自动识别并选择最优策略！
```

## 🔧 故障排除类

### Q8: 任务执行失败怎么办？
**回复：**
```
排查步骤：
1. 检查 Gateway 状态：openclaw status
2. 查看任务日志（operations 目录）
3. 确认子 Agent 配对授权：openclaw devices approve

如仍无法解决，请提供：
- 任务描述
- 错误信息
- 系统版本

我们会尽快协助处理！
```

### Q9: 如何查看日志？
**回复：**
```
日志位置：
- 系统日志：OpenClaw 默认日志目录
- 任务日志：skills/dynamic-multi-agent-system/operations/
- Token 监控：TOKEN-MONITOR.md

建议开启监控大屏实时查看任务状态。
```
