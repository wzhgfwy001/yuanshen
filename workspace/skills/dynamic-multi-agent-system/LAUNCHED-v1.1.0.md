## ✨ v1.1.0 核心亮点

### 🎯 8 大 Token 优化模块

1. **Token 缓存** - 避免重复调用（30-60% 节省）
2. **智能模型选择** - 按需分配最优模型（30-50% 节省）
3. **批量处理器** - 合并相似任务（50-70% 节省）

7. **结果复用** - 复用历史结果（50-60% 节省）
8. **可视化监控** - 实时追踪任务状态
