# 架构说明

## 整体架构

```
┌─────────────────────────────────────────────────────────┐
│                    用户 (User)                          │
│                      ↓ ↑                                │
│              主Agent (单一常驻)                          │
│  ┌───────────────────────────────────────────────────┐  │
│  │ 九大核心组件                                       │  │
│  │ 1.多任务队列管理器  2.任务分类器  3.任务分解器    │  │
│  │ 4.队伍组建器      5.执行协调器  6.质量检查器      │  │
│  │ 7.Skill进化分析器  8.资源清理器  9.反思改进器    │  │
│  └───────────────────────────────────────────────────┘  │
└───────────────────────────┬─────────────────────────────┘
                            │
        ┌───────────────────┼───────────────────┐
        ▼                   ▼                   ▼
┌───────────────┐   ┌───────────────┐   ┌───────────────┐
│  主任务A      │   │  主任务B      │   │  主任务C      │
│  (创新任务)   │   │  (标准任务)   │   │  (单一任务)   │
└───────┬───────┘   └───────┬───────┘   └───────┬───────┘
        │                   │                   │
        ▼                   ▼                   ▼
┌───────────────┐   ┌───────────────┐   ┌───────────────┐
│ 动态子Agent   │   │ 加载Skill     │   │ 主Agent直接   │
│ 团队          │   │ 固化流程      │   │ 执行          │
└───────┬───────┘   └───────┬───────┘   └───────┬───────┘
        │                   │                   │
        └───────────────────┴───────────────────┘
                            │
                            ▼
              ┌─────────────────────────┐
              │    共享状态层           │
              │  - skill-counters.json  │
              │  - experience-db.json   │
              │  - execution-logs/      │
              └─────────────────────────┘
```
