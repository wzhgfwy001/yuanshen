# xiaohongshu-editor

**【暗影摄怪】Shadow Craft** — 基于 Agency Agent 体系

## 功能

将原始内容改写为小红书风格的笔记：
- 标题优化（吸引眼球）
- 内容排版（分段、emoji）
- 标签添加（#话题）
- 行动号召（引导互动）

## 架构

本技能已整合到 **Agency Agent 体系**，使用 `marketing-xiaohongshu-specialist` 的专业知识：

```
xiaohongshu-editor（入口）
    │
    └── agency-registry/loader.ts
              │
              └── marketing-xiaohongshu-specialist.md
```

## 小红书内容标准（来自 Agency Agent）

### 内容比例

| 类型 | 比例 | 说明 |
|------|------|------|
| 有机生活内容 | 70% | 真实生活分享 |
| 趋势热点 | 20% | 跟随时事热点 |
| 品牌/产品 | 10% | 软性植入 |

### 爆款标题公式

```
疑问型：[人群] + [痛点] + [疑问]
数字型：[数字] + [效果] + [人群]
对比型：[A] vs [B]
情绪型：[情绪词] + [场景]
```

### 社区互动规则

- 使用热门话题标签（#探索 #穿搭 #美食 #旅行）
- 发布时间：工作日 12:00-13:30、19:00-21:00
- 互动指标：5%+ engagement rate 为优质

## 改写流程

```
原文 → 分析核心 → 重新选题 → 写标题 → 写开头 → 写正文 → 添加标签 → 检查
```

### 步骤说明

1. **提取核心观点** - 从原文中提炼关键信息
2. **设计爆款标题** - 使用上述公式
3. **重新组织内容** - 按模板结构输出
4. **添加emoji装饰** - 适量点缀，每段1-2个
5. **添加热门标签** - 3-10个相关话题
6. **互动引导** - 结尾引发评论

## 改写检查清单（来自 Agency Agent）

- [ ] 保持70%有机生活内容比例
- [ ] 标题引发好奇，点击率目标 > 5%
- [ ] 开头3秒抓住注意力
- [ ] 内容分点清晰，每段短小精悍
- [ ] emoji使用适量，不喧宾夺主
- [ ] 标签相关且热门
- [ ] 结尾有互动引导（问题/投票）
- [ ] 字数在300-1000字之间
- [ ] 无敏感词/违禁词
- [ ] 发布时间符合最佳时段

## 工具函数

本Skill提供以下工具函数：

```javascript
const xiaohongshu = require('./xiaohongshu-editor.js');

// 生成标题
xiaohongshu.generateTitle('时间管理', { style: 'practical' })

// 生成内容
xiaohongshu.write('时间管理技巧', {
  type: 'experience',  // experience | tutorial | review
  words: 500,
  tags: ['成长', '职场', '效率']
})

// 生成系列选题
xiaohongshu.generateSeries('护肤', 5)
```
