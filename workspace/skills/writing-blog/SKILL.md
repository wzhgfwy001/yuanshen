Use this skill when user asks to write a blog post or article.

- ✅ 所有功能完全可用

## Blog Writing Process

1. **Understand the topic** - Clarify audience and purpose
2. **Create outline** - Structure with intro, body, conclusion
3. **Write engaging intro** - Hook the reader
4. **Develop body** - 3-5 key points with examples
5. **Strong conclusion** - Call to action or summary

## Template Structure

```markdown
## [Attention-Grabbing Title]

[Hook sentence - problem or question]

### [Section 1: Why This Matters]
[Practical content with examples]

### [Section 2: Key Insight 1]
[Actionable advice]

### [Section 3: Key Insight 2]
[Actionable advice]

### [Section 4: How to Apply]
[Step-by-step guide]

## [Conclusion with CTA]
[Summary + next steps]
```

## Blog Types

| Type | Style | Length |
|------|-------|--------|
| Tutorial | Step-by-step | 1500-3000字 |
| Listicle | Numbered points | 800-1500字 |
| Opinion | Personal perspective | 500-1000字 |
| Review | Feature + verdict | 1000-2000字 |

## Quality Checklist

- [ ] Title is compelling
- [ ] Hook grabs attention in first sentence
- [ ] Each section has a clear point
- [ ] Includes concrete examples
- [ ] Conclusion has clear CTA
- [ ] SEO keywords naturally integrated
