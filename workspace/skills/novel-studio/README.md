# 小说创作助手 - Novel Studio

**类型：** 行业套件

**版本：** v1.0.0

---

## 功能

通过5阶段引导，帮助用户构建完整小说：
1. 世界观构建
2. 剧情架构
3. 人物塑造
4. 正文撰写
5. 审查润色

## 适用类型

科幻、奇幻、言情、悬疑、都市等

## 使用方式

```
写小说 → 启动小说创作助手
继续写作 → 从上次中断处继续
```

## Agent分工

- 山寺三郎：世界观设计、正文撰写
- 波特卡斯·D·艾斯：科幻元素分析
- 奈良鹿丸：剧情架构
- 朽木白哉：人物塑造
- 质量检查Agent：最终审查

## 优势

- 引导式构建，不用从零开始
- 多Agent协作，质量更高
- 5阶段质检
- SKILL可固化
