# Standing Orders - 持久化指令

这些指令在每次会话中自动加载，我会在指定条件下自动执行。

---

## Program: 每日早间简报

**授权:** 汇总信息，生成简报
**触发:** 每天早上8:00 (Cron配合)
**审批:** 不需要，每次自动执行

### 执行步骤
1. 检查今日日程（从记忆系统读取）
2. 检查待处理任务（brain/tasks/active.md）
3. 检查inbox是否有紧急事项
4. 生成简报输出给我

### 简报格式
```
🌅 早间简报 - {日期}

📅 今日日程：
- [日程项]

📋 待办任务：
- [任务项]

📬 待处理：
- [inbox中的重要事项]
```

## Program: 磁盘空间监控

**授权:** 检查磁盘空间，低于阈值时提醒
**触发:** 每6小时自动检查
**审批:** 不需要，监控类

### 执行步骤
1. 检查系统磁盘空间
2. 如果C盘可用空间 < 10GB
3. 生成告警报告

### 阈值
- 警告：< 10GB
- 严重：< 5GB
