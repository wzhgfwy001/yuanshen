# 银魂风格海报 prompt

## 基于歌曲 "Life Goes On" 的银魂风格海报

### 画面内容
- 主要角色：银魂风格的坂田银时（或类似形象）
- 场景：破旧但温馨的小酒馆或小公寓
- 元素：
  - 空的钱包/口袋（但里面有桃花）
  - 窗外的朝阳/希望之光
  - 墙上有"MONEY"字样但被花覆盖
  - 简单但温暖的室内场景

### 风格
- 银魂(Gintama)动漫风格
- 典型的银魂幽默感：穷人但乐观
- 有"丧而不伤"的日漫美学
- 角色表情：虽然窘迫但自信微笑

### 构图
- 中央：主要角色半身像
- 背景：破旧但温馨的房间
- 前景：散落的钱币（很少）和小花
- 色调：暖色调为主，表示希望

### 文字（可选）
- "Life Goes On" 或 "まだ終わらない"
- 底部小字："人は皆 傷つきながら前行く"

### 日文提示词

- 早晨阳光透过窗户
