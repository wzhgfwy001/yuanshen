## 🔍 问题诊断

### 1. AI味太重

**表现：**
- 描写太完美、太套路
- 对话像在"演讲"，不够口语化
- 人物反应太一致，缺乏个性
- 情绪表达夸张，不够真实

**例子：**
```
❌ AI味："凌午夜的眼中闪过坚定的光芒，那是不屈的意志..."
✅ 更真实："凌午夜没说话，只是攥紧了拳头。"
```

### 2. 逻辑不缜密

**问题：**
- 时间线不清晰
- 力量体系有漏洞
- 势力关系矛盾
- 地理设定模糊

### 3. 人物称呼混乱

**问题：**
- 有时叫"东方凌薇"，有时叫"凌薇"
- 南宫墨的家人称呼不确定
- 势力内部怎么称呼没统一

## ✍️ 去AI味技巧

### 1. 对话要"不完整"
```
❌ AI："我理解你的感受，让我们一起面对困难！"
✅ 真实："......"（沉默有时比说话更有力）
```

### 2. 描写要"不完美"
```
❌ AI："他的眼中闪烁着坚定的光芒。"
✅ 真实："他低着头，手在发抖。"
```

### 3. 反应要"有个性"
```
凌午夜：沉默/直接/行动派
东方凌薇：理性/分析/但关键时刻感性
南宫墨：冷静/毒舌/实际
```

### 4. 场景要"有细节"
```
❌ AI："教室里弥漫着紧张的气氛。"
✅ 真实："教室很安静。有人在喝水，有人在转笔。"
```
