# 决策：记忆系统优化方案

**日期：** 2026-04-07
**决策类型：** 系统架构
**状态：** 已执行

## 决策内容

采用Loci风格的记忆系统优化方案，创建brain/目录结构。

## 决策原因

1. 现有MEMORY.md过于臃肿，查找效率低
2. 没有层级意识，所有信息混在一起
3. 没有自动化的信息流动规则
4. 缺乏任务追踪和决策归档

## 选择方案

创建brain/目录，包含：
- plan.md（人生方向）
- me/（身份信息）
- tasks/（任务管理）
- decisions/（决策归档）
- inbox.md（收件箱）

## 不选择其他方案的原因

- Loci方案成熟，有明确的信号→目的地规则
- 与现有memory/系统兼容，不冲突
- 与多Agent系统隔离清晰，职责分明
