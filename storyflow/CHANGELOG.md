# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.2.0] - 2026-04-20

### Added
- storyflow 5-Agent workflow support
- 33-dimensional quality audit system
- AI trace detection and removal
- Truth files driven workflow
- RadarNode, ArchitectNode, WriterNode, AuditorNode, ReviserNode

### Enhanced
- Improved loop engine for iterative refinement
- Enhanced checkpoint system
- Better error handling and logging

## [1.1.0] - 2026-04-15

### Added
- Multi-model provider support (MiniMax, Tongyi, Claude)
- Environment variable API key configuration
- Checkpoint and cache mechanism
- Web server with Flask

### Enhanced
- Workflow visualization with React Flow
- CLI interface improvements

## [1.0.0] - 2026-04-10

### Added
- Initial release
- Basic workflow engine
- WorldBuilding, Character, Chapter generation nodes
