## 🎯 项目概述

StoryFlow 是一个节点化小说生成工作流系统，借鉴 ComfyUI 的思想，将小说创作拆解为可复用的节点。

### 核心特性
- ✅ 节点化工作流（可自由组合）
- ✅ 拓扑排序执行（自动确定执行顺序）
- ✅ 并行执行支持（提升效率）
- ✅ 循环依赖检测（避免死锁）
- ✅ 完整错误处理（重试机制、检查点）
- ✅ 状态管理系统（暂停/恢复/回滚）

### 技术栈
- **后端**: Python 3.10+ + FastAPI
- **前端**: React Flow + D3.js
- **任务队列**: Celery + Redis
- **LLM**: 通义千问 API
