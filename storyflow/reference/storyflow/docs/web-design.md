# StoryFlow Web 版本设计文档

## 概述

StoryFlow Web 是一个可视化工作流编辑器，采用基于节点的编程范式，灵感来自 ComfyUI。设计目标是直观、强大、易于扩展。

## 功能模块

### 1. 顶部工具栏

```
┌─────────────────────────────────────────────────────────────────────┐
│ 📄 StoryFlow  [新建 ▼] [打开] [保存] [导出 ▼]    🔍 搜索节点  ⚙️  │
└─────────────────────────────────────────────────────────────────────┘
```

**功能说明：**

- **新建**：创建新工作流
  - 空白工作流
  - 从模板创建
  - 导入现有工作流

- **打开**：打开已保存的工作流
  - 最近使用的列表
  - 本地文件浏览
  - 从 URL 导入

- **保存**：保存当前工作流
  - 快捷键：Ctrl+S / Cmd+S
  - 自动保存：每 5 分钟
  - 保存为模板

- **导出**：导出工作流
  - YAML 格式
  - JSON 格式
  - PNG 图片（当前视图）
  - 分享链接

- **搜索**：快速搜索节点
  - 快捷键：Ctrl+Shift+N / Cmd+Shift+N
  - 支持模糊匹配
  - 显示最近使用

- **设置**：全局设置
  - API 配置
  - 主题设置
  - 键盘快捷键
  - 插件管理

#### 节点状态

- **默认状态**：白色背景，黑色边框
- **选中状态**：蓝色边框，阴影效果
- **运行中**：边框闪烁动画，进度条
- **完成状态**：绿色边框 ✓
- **错误状态**：红色边框 ✗
- **禁用状态**：灰色，半透明
- **折叠状态**：仅显示标题栏

#### 节点端口

- **输入端口**（左侧 🔵）：
  - 蓝色：文本输入
  - 紫色：数字输入
  - 橙色：文件输入
  - 绿色：布尔值输入

- **输出端口**（右侧 🟢）：
  - 绿色：文本输出
  - 紫色：数字输出
  - 橙色：文件输出
  - 青色：多值输出

### 5. 连线交互

#### 创建连线

1. **鼠标操作**：
   - 点击输出端口（右侧）
   - 拖拽到目标节点的输入端口
   - 松开鼠标完成连接

2. **视觉反馈**：
   - 连线跟随鼠标移动
   - 目标端口高亮显示
   - 不兼容的端口显示红色禁止符号

#### 连线样式

```
正常连线: ──────────────  (灰色，实线)
激活连线: ──────────────  (蓝色，实线)
错误连线: ═══════════════  (红色，虚线)
可选连线: - - - - - - - -  (灰色，虚线)
```

#### 贝塞尔曲线

连线使用三次贝塞尔曲线，确保平滑美观：

```
源端口 (x1, y1)
    \
     \ 控制点 1 (x1 + 50, y1)
      \
       ◞──────────◟
      / 控制点 2 (x2 - 50, y2)
     /
目标端口 (x2, y2)
```

#### 连线操作

- **点击连线**：选中连线
- **右键连线**：显示选项
  - 删除连线
  - 禁用连线
  - 添加标签
- **拖拽连线中点**：调整连线形状
- **删除连线**：选中后按 Delete 键

#### 连线验证

- **类型检查**：确保输入/输出类型兼容
- **循环检测**：防止创建循环依赖
- **多连接检查**：
  - 输入端口：只能连接一条连线
  - 输出端口：可以连接多条连线

**错误提示：**

```
✗ 无法连接
  原因: 类型不匹配
  源类型: text
  目标类型: number

提示: 使用"文本转数字"节点进行转换
```

## 性能优化

### 1. 虚拟化渲染

- 画布使用虚拟滚动，只渲染可见区域
- 大量节点时自动简化非活跃节点
- 连线使用 SVG 路径优化

### 2. 增量更新

- 只更新变更的节点
- 连线缓存复用
- 属性面板按需加载

### 3. 请求优化

- 并行执行无依赖节点
- API 请求批量合并
- 智能缓存中间结果

### 4. 响应式设计

- 自适应窗口大小
- 移动端优化（触摸手势）
- 低性能模式可选

    "my-custom-node": {
      inputs: ["text"],
      outputs: ["text"],

        // 执行逻辑

    }
  },

  // 自定义UI

};
