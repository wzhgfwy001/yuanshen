## 📝 使用示例

### 1. 创建工作流
```python

from nodes import WorldBuildingNode

workflow = Workflow("my_workflow", "我的工作流")

node.set_input("genre", "奇幻")
workflow.add_node(node)
```

### 2. 执行工作流
```python
engine = Engine(workflow)
result = await engine.execute()
```

### 3. 查看结果
```python
for node_id, node_result in result["results"].items():
    print(f"{node_id}: {node_result}")
```

## 🔄 可扩展性

### 轻松添加新节点类型
```python
class MyCustomNode(LLMNode):
    def __init__(self, node_id: str, api_key: str, model: str):
        super().__init__(node_id, "我的节点", api_key, model)
        self.add_input("prompt", "str")
        self.add_output("response", "str")

    async def execute(self) -> NodeResult:
        prompt = self.input_values.get("prompt", "")
        response = await self.call_llm(prompt)
        return NodeResult(success=True, data={"response": response})
```

- 多分支并行
- 条件判断（通过节点逻辑）
- 数据聚合
- 循环处理（注意避免循环依赖）
