# Excel 筛选查找器 v1.3

高考志愿填报辅助工具 - 根据选科条件和分数筛选可报考的学校和专业。

## 🚀 v1.3 新功能

- 🔍 **学校搜索功能** - 实时搜索，智能提示，多选支持
- 📊 **Excel导出增强** - 完整数据导出，中文列名
- 📄 **PDF导出功能** - 美观报告，适合打印
- 💾 **数据持久化** - SQLite数据库，筛选历史保存

## 功能特点

- ✅ 支持导入 Excel 数据（学校专业表、学生信息表）
- ✅ 智能选科匹配（必选/可选科目）
- ✅ 分数/排名双重匹配
- ✅ 冲稳保三档自动划分
- ✅ 多条件组合筛选（地域、学校、专业）
- ✅ 结果导出 Excel（完整功能）
- ✅ 结果导出 PDF（新增）
- ✅ 筛选历史保存（新增）
- ✅ 学校实时搜索（新增）

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
# 方法1: 直接运行
python src/main.py

# 方法2: 使用启动脚本
python run_app.py
```

### 打包为 EXE

```bash
pyinstaller --onefile --windowed --name "Excel 筛选查找器" src/main.py
```

## 数据格式

### 学校专业表 (schools.xlsx)

| 字段 | 说明 | 示例 |
|------|------|------|
| major_id | 专业 ID | MAJ001 |
| school_id | 学校 ID | SCH001 |
| school_name | 学校名称 | 清华大学 |
| major_name | 专业名称 | 计算机科学与技术 |
| major_category | 专业类别 | 理工 |
| subject_requirement | 选科要求 | 物理 + 化学 |
| subject_required | 必选科目 | 物理，化学 |
| subject_optional | 可选科目 | 生物 |
| year | 招生年份 | 2025 |

| min_rank | 最低排名 | 100 |

| 字段 | 说明 | 示例 |
|------|------|------|
| student_id | 学生 ID | STU001 |

| category | 考生类别 | 物理类 |

## 使用说明

### 基本流程
1. **导入数据**：点击"导入学校数据"和"导入学生 Excel"
2. **选择学生**：从下拉列表中选择学生
3. **设置条件**：
   - 地域筛选（多选省份）
   - 学校搜索（输入关键词搜索）
   - 专业筛选（多选专业类别）
   - 学校层次（985/211/双一流等）
   - 匹配类型（冲/稳/保）
4. **开始筛选**：点击"开始筛选"按钮
5. **查看结果**：右侧显示匹配的学校和专业
6. **导出结果**：
   - 📥 Excel导出 - 完整数据表格
   - 📄 PDF导出 - 美观打印报告
   - 💾 保存历史 - 记录筛选结果

### 高级功能
- **学校搜索**：输入学校名称关键词实时搜索
- **多选学校**：可同时选择多个目标学校
- **筛选历史**：自动保存每次筛选记录
- **分页浏览**：大量结果分页显示（100条/页）

## 冲稳保规则

- **冲**：排名落后 500 名以内（可冲刺）
- **稳**：排名持平或领先（较稳妥）
- **保**：排名领先 1000 名以上（保底）
