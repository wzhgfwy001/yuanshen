## 🚀 使用流程

### 方式 1：手动输入学生信息

1. 打开程序
2. 点击"选择学生" → "手动输入"
3. 填写姓名
4. 勾选选科科目（至少 1 门）
5. 输入各科分数
6. **总分自动计算**
7. 填写排名（可选）
8. 点击"保存"
9. 开始筛选

### 方式 2：Excel 导入（原有功能）

1. 点击"导入学生 Excel"
2. 选择学生 Excel 文件
3. 选择要筛选的学生
4. 开始筛选
