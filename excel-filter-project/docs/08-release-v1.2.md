## 🎯 使用流程（完整版）

### 步骤 1：录入学生信息

**方式 A：手动输入（推荐）**
1. 打开程序 → 点击"🎓 打开学生信息录入系统"
2. 填写姓名
3. 勾选选科科目（至少 1 门）
4. 输入各科分数
5. **总分自动计算（绿色大字显示）**
6. 填写排名（可选）
7. 点击"✅ 保存信息"

**方式 B：Excel 导入**
1. 点击"导入学生 Excel"
2. 选择学生 Excel 文件
3. 选择要筛选的学生

---

### 步骤 2：设置筛选条件

**非必填，可自由组合：**
1. 勾选地域（北京/上海/江苏等）
2. 搜索学校（可选）
3. 勾选专业（计算机/金融/医学等）
4. 选择学校层次（全部/985/211/双一流/普通本科）
5. 勾选匹配类型（冲/稳/保）
6. 点击"开始筛选"

---

### 步骤 3：查看筛选结果

**结果面板功能：**
1. **查看统计** - 总结果数、冲/稳/保分布
2. **分页浏览** - 每页 100 条，翻页查看
3. **查看详情** - 点击"📄 详情"查看完整信息
4. **导出 Excel** - 点击"📥 导出 Excel"保存结果
5. **打印** - 点击"🖨️ 打印"打印结果
