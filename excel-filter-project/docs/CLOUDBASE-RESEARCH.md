## 📋 微信云开发 vs CloudBase

| 对比项 | 微信云开发 | CloudBase |
|--------|-----------|-----------|
| 定位 | 微信小程序专用 | 多端（小程序/Web/App） |
| 免费额度 | 有限 | 更充足 |
| 控制台 | mp.weixin.qq.com | cloud.tencent.com |
| 数据库 | 微信私有协议 | 腾讯云MySQL/NoSQL |

**结论：** 对于高考志愿系统，两个都可以用，推荐**微信云开发**（更适合小程序生态）

## 📝 云函数代码结构

### filter 函数（筛选）
```javascript
// 云函数入口
exports.main = async (event, context) => {
  const { score, type, province } = event;
  const openid = cloud.getWXContext().OPENID;
  
  // 检查用户权限
  const user = await db.collection('users').where({ openid }).get();
  
  if (user.free_count <= 0 && user.paid_count <= 0) {
    return { code: -1, msg: '需要购买会员' };
  }
  
  // 执行筛选
  const result = await db.collection(type)
    .where({ province })
    .field({ school: true, major: true, score_2024: true })
    .get();
    
  return { code: 0, data: result };
};
```
