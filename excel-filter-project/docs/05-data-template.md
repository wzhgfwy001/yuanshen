# 数据模板说明

## 学校专业表模板 (schools.xlsx)

### 字段说明

| 字段名 | 类型 | 必填 | 说明 | 示例 |
|--------|------|------|------|------|
| major_id | 文本 | 是 | 专业唯一标识 | MAJ001 |
| school_id | 文本 | 是 | 学校唯一标识 | SCH001 |
| school_name | 文本 | 是 | 学校名称 | 清华大学 |
| major_name | 文本 | 是 | 专业名称 | 计算机科学与技术 |
| major_category | 文本 | 是 | 专业类别 | 理工/文史/医学/艺术 |
| subject_requirement | 文本 | 是 | 选科要求描述 | 物理 + 化学 |
| subject_required | 文本 | 否 | 必选科目（逗号分隔） | 物理，化学 |
| subject_optional | 文本 | 否 | 可选科目（逗号分隔） | 生物，地理 |
| year | 数字 | 是 | 招生年份 | 2025 |
| min_score | 数字 | 是 | 最低录取分 | 680 |
| min_rank | 数字 | 是 | 最低录取排名 | 100 |
| avg_score | 数字 | 否 | 平均录取分 | 685 |
| enrollment_count | 数字 | 否 | 招生人数 | 50 |
| tuition | 数字 | 否 | 学费（元/年） | 5000 |

### 选科要求格式说明

| 要求类型 | subject_required | subject_optional | 说明 |
|----------|-----------------|-----------------|------|
| 不限 | (空) | (空) | 任何选科都可报考 |
| 必选单科 | 物理 | (空) | 必须选物理 |
| 必选双科 | 物理，化学 | (空) | 物理和化学都必须选 |
| 必选 + 可选 | 物理 | 化学，生物 | 必选物理，且化学/生物至少选一门 |
| 多选一 | (空) | 物理，化学，生物 | 物理/化学/生物至少选一门 |

## 数据准备指南

### 步骤 1：收集官方数据

1. 访问本省教育考试院官网
2. 下载往年高校录取数据
3. 整理成上述格式

### 步骤 2：数据清洗

1. 检查必填字段是否完整
2. 统一学校名称（避免"清华大学"和"清华"混用）
3. 统一科目名称（避免"物理"和"物理学"混用）
4. 检查分数和排名是否为数字

### 步骤 3：数据验证

1. 用 Excel 打开，检查格式
2. 导入程序测试
3. 抽查部分学校验证准确性

### 步骤 4：数据更新

1. 每年更新录取数据
2. 保留历史数据（添加 year 字段区分）
3. 备份旧版本数据

## 常见问题

### Q1: 选科要求如何填写？

**A:** 根据学校官方招生简章填写：
- "物理 + 化学"表示两科必选
- "物理/化学"表示选一科即可
- "不限"表示无选科要求

### Q2: 排名是位次还是分数段？

**A:** 使用最低录取位次（如最低分对应的全省排名）

### Q3: 多年数据如何处理？

**A:** 每条记录添加 year 字段，筛选时可选择年份

### Q4: 中外合作办学专业如何标注？

**A:** 在专业名称后加括号注明，如"计算机科学与技术 (中外合作办学)"
