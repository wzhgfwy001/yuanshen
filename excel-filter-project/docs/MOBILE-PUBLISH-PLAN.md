# 山东高考志愿填报系统 - 移动端方案

## 📋 项目概述

**现有系统：** Python Tkinter 桌面应用
- main_launcher.py（主界面）
- main_simple2.py（本科筛选器）
- main_zhuanke.py（专科筛选器）
- 数据文件：山东省2024-2025高考数据

**目标：** 开发微信小程序 + Android + iOS 应用

---

## 方案一：微信小程序（推荐起步）

### 为什么首选小程序？
- 开发成本低（使用Web技术）
- 无需版号申请
- 微信生态，触达用户快
- 国内高中生必备微信

### 技术架构

```
┌─────────────────────────────────────┐
│         微信小程序前端               │
│   (WXML + WXSS + JS / Vue框架)      │
├─────────────────────────────────────┤
│         云开发 / 独立服务器          │
│   (用户数据 + 院校数据库)            │
├─────────────────────────────────────┤
│         数据同步接口                 │
│   (Python后台 → JSON API)           │
└─────────────────────────────────────┘
```

### 技术选型

| 方案 | 技术栈 | 开发周期 | 成本 |
|------|--------|----------|------|
| 原生开发 | 微信开发者工具 | 2-3个月 | 3-5万 |
| Flutter跨平台 | Flutter + 后端API | 2-3个月 | 4-6万 |
| Taro跨平台 | Taro + React | 2-3个月 | 3-5万 |

### 小程序上架流程

```
1. 注册微信小程序账号
   ↓
2. 完成主体认证（个人/企业）
   ↓
3. 开发小程序（2-3个月）
   ↓
4. 配置服务器域名
   ↓
5. 提交审核（7个工作日内）
   ↓
6. 发布上线
```

### 需要准备的材料

**个人开发者：**
- 身份证
- 微信账号
- 手机号

**企业开发者：**
- 营业执照
- 对公账户
- 法人身份证
- 微信服务号（可选）

### 费用

| 项目 | 费用 |
|------|------|
| 微信小程序注册 | 免费 |
| 主体认证（个人） | 免费 |
| 主体认证（企业） | 300元/年 |
| 开发成本 | 0（自研）~ 5万（外包） |

| 360手机助手 | 免费 | 3-5天 | 低 |
| Google Play | 25美元 | 1-3天 | 低 |

### Android开发方案

**推荐：Flutter跨平台开发**

```
Python桌面应用 → Flutter移动应用
                      ↓
         同时支持 Android + iOS
```

**Flutter优势：**
- 一套代码，生成两个平台
- 性能接近原生
- 开发周期短（2-3个月）
- 社区活跃

### Android上架材料清单

```
1. 软件著作权证书（必须！）
   - 费用：600-1000元
   - 周期：1-2个月
   
2. 企业营业执照 或 个人身份证

3. 应用图标（1024x1024 PNG）

4. 应用截图（多尺寸）

5. 应用描述文案

6. 隐私政策URL（必须！）

7. ICP备案（服务器需备案）
```

### Android上架流程

```
1. 申请软件著作权（1-2个月）
   ↓
2. 注册开发者账号
   - 应用宝：https://open.qq.com/
   - 华为：https://developer.huawei.com/
   ↓
3. 开发/打包APK
   ↓
4. 提交审核材料
   ↓
5. 应用市场审核（3-7天）
   ↓
6. 发布上线
```

### 费用估算

| 项目 | 费用 | 备注 |
|------|------|------|
| 软件著作权 | 600-1000元 | 必须 |
| 开发者账号 | 免费（大部分） | 企业需认证 |

## 方案三：iOS应用

### 上架平台

- **App Store**（必须）

### 必需材料

```
1. Apple开发者账号
   - 个人：688元/年
   - 企业：688元/年
   
2. 软件著作权（强烈建议）

3. 应用图标（1024x1024 PNG）

4. 应用截图（iPhone尺寸）

5. 隐私政策URL

6. 支持页面URL
```

### iOS上架流程

```
1. 申请Apple开发者账号（688元/年）
   ↓
2. 创建App ID
   ↓
3. 创建应用条目（App Store Connect）
   ↓
4. 开发打包（Xcode → .ipa）
   ↓
5. 提交审核（1-7天）
   ↓
6. 审核通过 → 发布
```

### 审核重点

- 崩溃Bug（必须0崩溃）
- 加载时间（不超过3秒）
- 隐私合规
- 内容合规

| 软件著作权 | 600-1000元 |
| Mac电脑（如需自研） | 5000元+ |
| 开发成本 | 0（自研）~ 10万 |
