### 1.2 用户场景分析

#### 场景 1：高考志愿填报（主要场景）
- **用户**: 高三学生或家长
- **痛点**: 手动筛选大量数据耗时耗力，容易遗漏
- **解决方案**: 一键筛选，自动冲稳保分类

#### 场景 2：模拟考试志愿预演
- **用户**: 高三学生（一模、二模后）
- **痛点**: 不了解当前成绩能报什么学校
- **解决方案**: 输入模拟成绩，查看目标差距

#### 场景 3：升学规划咨询
- **用户**: 高中老师、升学顾问
- **痛点**: 需要为多个学生提供建议
- **解决方案**: 批量导入学生，批量生成报告

### 2.3 核心算法

#### 选科匹配算法
```python
def match_subjects(student_subjects, required, optional):
    # 必选科目必须全部满足
    if required:
        for subj in required:
            if subj not in student_subjects:
                return False
    # 可选科目：至少选一科
    if optional:
        if not any(subj in student_subjects for subj in optional):
            return False
    return True
```

#### 冲稳保划分算法
```python
def calculate_tier(student_score, admit_score, student_rank, admit_rank):
    score_gap = student_score - admit_score
    rank_gap = admit_rank - student_rank
    
    if rank_gap >= 1000:   # 领先 1000 名以上
        return '保'
    elif rank_gap >= 0:    # 排名持平或领先
        return '稳'
    elif rank_gap >= -500: # 落后 500 名以内
        return '冲'
    else:
        return '不匹配'
```
